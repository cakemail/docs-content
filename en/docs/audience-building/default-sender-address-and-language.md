# Default Sender Address and Language

## Description

Default sender address and language settings control how your automated welcome and confirmation emails appear to new subscribers. This feature allows you to customize language preferences for all system-generated emails, including subscription confirmations, welcome messages, and unsubscription confirmations. Proper configuration ensures these automated touchpoints align with your brand identity and provide a consistent experience across all subscriber interactions.

## Why It's Important

To build your brand and be recognizable by recipients, you should choose the right language to be recognized by your recipients.

## Step-by-Step Instructions

![Set default sender and language](/images/forms/Nov-09-2021_11-11-00.gif)

Follow these simple steps to properly specify the default sender for your list and its automated confirmation emails:

<Stepper>

1. Select the **Contacts** tab
![Select the Contacts tab](/images/forms/B7680B8E-0822-4DE1-A29F-A3EEB9FE5DB8_4_5005_c.jpeg)

1. Choose your contact list from the dropdown menu
![Choose contact list](/images/forms/ACDED713-00D2-45DD-92E8-BA34863B7A27_4_5005_c.jpeg)

1. Click the **gear icon** to access the list management menu
![Click the gear icon](/images/forms/CD6FF683-4314-4B46-A481-B23A2F173071_4_5005_c.jpeg)

1. On the list manager menu, click the **Settings** tab
![Click the Settings tab](/images/forms/BA70BCD2-FF6F-4263-9E74-5E1179E5F04F_4_5005_c.jpeg)

1. Select your sender from the **Default Sender** option
![Select default sender](/images/forms/04506F34-5070-40C0-A476-FD33B38C9515_4_5005_c.jpeg)
You can add your sender email if you don't find it in the list by selecting **Manage Senders**.
*Reference: Adding a new sender*

1. Finally, you can set the default language for the list from this menu. Select the **Settings** tab to set the **Default Language** for the list
![Set default language](/images/forms/882BB14B-BD5B-4ACF-BACD-CB920E8E686A.png)
This is the default language users will see when managing their subscriptions. This will not affect the language of your email campaigns.
</Stepper>

## Troubleshooting

**I can't find my sender in the dropdown list**

- Click "Manage Senders" to add a new sender email
- Verify the sender email has been confirmed
- Check if you have permissions to manage senders
- Make sure the sender isn't archived or deleted

**Language settings aren't applying correctly**

- Confirm you saved changes after selecting the language
- Check if browser language is overriding settings
- Verify the selected language is supported
- Test with a new subscriber to see current settings

**Welcome emails display wrong information**

- Review that sender and language settings are saved
- Check if there are conflicting settings at the account level
- Make sure template language matches default language
- Verify changes have propagated (may take a few minutes)
