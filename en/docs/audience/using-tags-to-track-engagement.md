# Using Tags to Track Engagement

## Description

You can create segments based on actions by tagging contacts. When you tag a contact, you update their profile in real-time. For example, if your subscribers click on certain email links, you can tag them with specific labels.

In essence, your email contacts can be classified according to your segmentation needs by creating tags. You can also tag subscribers based on other actions, like how they interact with your products. We'll show you how to create and use tags in this article, as well as how to do it for your entire contact list.

## Why It's Important

Tags are a powerful tool for advanced segmentation in email marketing. They allow you to categorize contacts based on their behavior, preferences, and interactions with your campaigns in real-time. This granular level of organization enables you to send highly targeted campaigns to specific groups, improving engagement rates and conversions. Tags also help you track customer journey stages, automate workflows based on subscriber actions, and maintain a clean and well-organized contact database that scales with your business.

## Step-by-Step Instructions

Here's how to create and use a tag.

<Stepper>

1. Click on your username in the top right corner and select **Tag Library** from the dropdown menu
![Access tag library from user menu](/images/manage-contacts/manage-contacts-creating-and-using-tags-38.png)

1. Enter the name of your new tag
![Enter new tag name in text field](/images/manage-contacts/manage-contacts-creating-and-using-tags-39.png)

1. Click **Create 1 tag**
![Click Create 1 tag button](/images/manage-contacts/manage-contacts-creating-and-using-tags-40.png)

1. Your tag is now created! You can close the menu
![Tag created confirmation screen](/images/manage-contacts/manage-contacts-creating-and-using-tags-41.png)
</Stepper>

## How to Tag a Contact

To use a tag, follow these steps:

<Stepper>

1. Select the **Contacts** tab
![Navigate to Contacts tab](/images/manage-contacts/manage-contacts-creating-and-using-tags-42.png)

1. Select your contact list
![Select your contact list from available lists](/images/manage-contacts/manage-contacts-creating-and-using-tags-43.png)

1. Check the circle next to the email address
![Select contact by checking the box](/images/manage-contacts/manage-contacts-creating-and-using-tags-44.png)

1. The bottom of the page will display a new menu. Select **Add or remove tags** from the Actions menu
![Select Add or remove tags from Actions menu](/images/manage-contacts/manage-contacts-creating-and-using-tags-45.png)

1. Choose the tag you want to use and click **Tag**. Alternatively, you can create a tag directly in this menu
![Select tag and click Tag button](/images/manage-contacts/manage-contacts-creating-and-using-tags-46.png)

1. When you've selected the tag, click **Done**
![Click Done button to confirm tag selection](/images/manage-contacts/manage-contacts-creating-and-using-tags-47.png)

1. It's that simple, your contact is now tagged!
![Successfully tagged contact confirmation](/images/manage-contacts/manage-contacts-creating-and-using-tags-48.png)
</Stepper>

## Segmenting Your Contacts When Importing a List

It's also possible to include a label when importing your contact list, thus automatically segmenting the list during import.

![Contact import interface with tags](/images/manage-contacts/manage-contacts-creating-and-using-tags-49.png)

If you already have a segment based on a label, you can select that segment during the import process, and each contact in the list will automatically be added to the segment.

You can also create a new label or add an existing label to all your contacts during the list import validation step for future use.

## How to Import Existing Tags
If you're already using tags in another email service, you can import your list with tags separated by ",". During the import process, tags will be created individually.

## Troubleshooting

**Tags don't appear in tag library**

- Refresh the page after creating tags
- Check for duplicate tag names
- Make sure tags were saved correctly
- Log out and log back in to refresh the view

**Unable to add tags to contacts**

- Check that contacts are selected (box checked)
- Verify tags exist in the library first
- Make sure you have permission to edit contacts
- Try tagging one contact at a time

**Tags not importing with contacts**

- Include tags in a dedicated CSV column
- Use comma separation for multiple tags
- Create tags in library before import
- Check that import mapping includes tag column

**Unable to create new tags**

- Check for special characters in tag names
- Make sure tag names aren't too long
- Verify you haven't reached tag limits
- Try creating tags directly from tag library

**Tags not working in segments**

- Allow time for tags to be processed after adding
- Make sure tags are spelled correctly in segments
- Check that tagged contacts are in the correct list
- Refresh segment after tagging contacts

**Bulk tagging fails**

- Reduce number of selected contacts
- Check for timeout issues with large batches
- Make sure all selected contacts are active
- Try tagging in smaller groups of 500 or less
