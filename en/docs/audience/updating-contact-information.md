# Updating Contact Information

## Description

Your contact list can be modified or updated at any time. You can, for example, enter an attribute, like a birthday, for a contact to differentiate and target that recipient. Also available are options to unsubscribe, delete, or add a contact to a suppression list.

## Why It's Important

Some contacts may need to be manually removed from your list to maintain a clean list, while others may simply need their contact information changed. The resulting challenges in finding valuable information in the constantly evolving and growing contact list make managing contact information essential to organizational efficiency.

## Troubleshooting

**Unable to find specific contact**

- Check if the email was typed correctly in search
- Try searching with a partial email address
- Make sure the contact is in the selected list
- Check if the contact might be unsubscribed

**Edit option doesn't appear**

- Make sure you clicked on the three dots menu (...)
- Check that you have permission to edit contacts
- Verify the contact isn't being processed
- Try refreshing the page

**Contact changes won't save**

- Check for validation errors in fields
- Make sure email format is valid
- Look for required fields left empty
- Try editing one field at a time

**Unable to add attributes to contact**

- Check that attributes exist for this list
- Verify attribute data type matches input
- Make sure date format is YYYY-MM-DD
- Clear any trailing spaces in text fields

**Bulk actions not working**

- Limit selection to 500 contacts at a time
- Make sure all selected contacts are active
- Check that you have necessary permissions
- Wait for previous bulk action to complete

**Contact shows wrong status**

- Allow time for status changes to update
- Check contact history for recent changes
- Verify contact isn't on suppression list
- Review unsubscribe logs for this contact
