# Exporting a Contact List

## Description

You can export your contact list as a .csv file for use in an external tool. This export will include the subscriber ID, address status, email address, and other attributes you have configured. Exported addresses will not be removed from the contact list.

## Plan Requirements

**Note:** List export is available exclusively to users subscribed to our **Growth or Premium plans**. Unlock this feature to easily download your data and have better control over your contacts.

## Why It's Important

Exporting your contact list can be a useful way to back up your data, analyze your audience, or transfer contacts to another platform. If you want to manage your lists outside of our application or reference them for cross-referencing, this function can be useful. When you launch a contact list export, the file will remain in your account settings for five days.

## What's Included in the Export

The exported CSV file will include:
- **Subscriber ID**
- **Address status** (active, unsubscribed, etc.)
- **Email address**
- **Custom attributes** you have configured
- **Subscription date** and other metadata

## Important Notes

- Exported addresses remain in your original contact list
- Export files are available for **5 days** after generation
- Available only on Growth and Premium plans
- Useful for data backup, analysis, or platform migration

## Troubleshooting

**Export option not available**

- Check that you have a Growth or Premium plan
- Verify your account permissions
- Make sure the list contains contacts to export
- Try selecting the list first before accessing Actions

**Export takes too long to process**

- Large lists can take 10-30 minutes
- Check your email for completion notification
- Try exporting a smaller segment first
- Make sure your account email is valid

**Unable to download exported file**

- Remember files expire after 5 days
- Check spam folder for download link
- Make sure browser allows downloads
- Try a different browser or disable extensions

**CSV file appears corrupted**

- Open with a spreadsheet program, not a text editor
- Check CSV delimiter settings (comma vs semicolon)
- Try importing into Google Sheets first
- Make sure special characters are UTF-8 encoded

**Missing data in export**

- Check that attributes were properly configured
- Verify segment criteria if exporting a segment
- Make sure data exists for exported fields
- Try exporting the full list instead of segment

**Export includes unwanted contacts**

- Carefully review segment criteria
- Check contact status filters
- Make sure you selected the correct list
- Consider creating a more specific segment
