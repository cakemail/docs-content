# Adding Attributes to Your List

## Description

You can use attributes to personalize your emails and create segments based on them. The goal is to produce more targeted emails using your subscribers' personal information. This approach offers individually tailored email marketing to each customer and substantially increases email marketing metrics. When new subscribers join your list, information related to these attributes can be collected.

## Why It's Important

In email marketing, personalization is undeniably king because it creates a stronger relationship between brand and customer, it's the best way to use the email program as a whole, and it offers customers a new experience.

## Step-by-Step Instructions

You have access to adding attributes in the list settings:

<Stepper>

1. Select the **Contacts** tab from the top menu of the dashboard
![Select the Contacts tab](/images/audience/C564B225-562B-474A-9FAB-E973EA758983_4_5005_c.jpeg)

1. Click on the **gear icon** on the right side of the page
![Click on the gear icon](/images/audience/265CA01A-E39B-48C5-BE11-0F35D63AA819_4_5005_c.jpeg)

1. At this point, an attribute can be added. Select the list to add the attribute name and choose its type: **string** or **date-time**
   
   **String:** This attribute is for any text field that can be used to characterize your contacts.
   - Examples: First name, last name, company, job title, city, preferences
   
   **Date-time:** With this attribute, you can specify a date, like a birthday.
   - Examples: Birthday, anniversary date, subscription date, last purchase date
   ![Screen_Shot_2021-05-26_at_11.31.41_AM.png](/images/audience/Screen_Shot_2021-05-26_at_11.31.41_AM.png)

1. You will find these attributes in the contact list and its subscription form that will be generated after adding them.
   
   Click on the... and Edit next to a contact to link the customer with an attribute.
   ![View attributes in contact list](/images/audience/EC6820BF-C35D-43C4-8347-28659FD3C89D_4_5005_c.jpeg)
   ![Edit contact attributes](/images/audience/Screen_Shot_2021-05-26_at_11.38.35_AM.png)
   
   By using this tool, you'll be able to segment your audience and optimize your personalization, resulting in a streamlined user experience.
   </Stepper>

## Using Attributes

Once created, these attributes can be used for:
- **Email personalization** with merge tags
- **Segmentation** to create targeted lists
- **Data collection** through subscription forms
- **Campaign personalization** based on subscriber data

## Benefits

- **Higher engagement** through personalized content
- **Better targeting** with segmented campaigns
- **Improved metrics** through relevant messaging
- **Enhanced customer experience** with tailored communications

## Troubleshooting

**Attribute doesn't appear in merge tags**

- Make sure the attribute was saved correctly
- Check that the attribute name doesn't contain special characters
- Refresh your browser or log out and log back in
- Check that the attribute is attached to the correct list

**Unable to create new attributes**

- Check your plan limits for custom attributes
- Make sure attribute names are unique within the list
- Avoid using reserved names like "email" or "id"
- Remove any leading or trailing spaces from attribute names

**Date-time attribute formatting issues**

- Use the correct format: YYYY-MM-DD
- Make sure dates are valid (e.g., not February 30th)
- Check timezone settings in your account
- Verify the date isn't in the future if it's a birth date

**Attributes not importing correctly**

- Check that CSV column headers match attribute names exactly
- Check for extra spaces or special characters in the CSV
- Make sure data types match (string vs date-time)
- Review import mapping during the import process

**Unable to delete or modify attributes**

- Check if the attribute is used in active campaigns
- Make sure you have the necessary permissions
- Try removing all data from the attribute first
- Contact support if the attribute is system-generated
