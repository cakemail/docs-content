# Adding PDF Attachments

## Attachments in Emails

There are several reasons why you shouldn't send emails with attachments: some ISPs may mark them as spam, subscribers may believe it's spam, and the message itself will be so large that it becomes a hassle for the recipient. We have a better alternative for you, one that is safer and more efficient.

## Attaching Files Properly

If you need to direct content to your message in order to overcome the obstacles mentioned above, we can host your file online, in our file manager, and add a link to your email content. You'll improve your deliverability and enhance the user experience for your recipients.

## Uploading Your PDF

First, import the PDF into the library, just as you would for an image. Here's how to proceed:

<Stepper>

1. Open the campaign.
![DD222A1A-C2FC-4946-97D9-77F6BB8BD302_4_5005_c.jpeg](/images/campaign-creation/campaigns-adding-a-pdf-to-a-campaign-270.jpeg)

1. In the email content, drag and drop a button.
![Image](/images/campaign-creation/mceclip0.png)

1. Click on the button, and in the right column under "Action", click "Link File"
![File link button in action menu](/images/campaign-creation/pdf.png)

1. Click Upload and browse. (PDFs are not visible in the library)
   ![Upload dialog for PDF files](/images/campaign-creation/campaigns-adding-a-pdf-to-a-campaign-273.jpeg)
   ![Screen_Shot_2021-07-05_at_2.18.49_PM.png](/images/campaign-creation/campaigns-adding-a-pdf-to-a-campaign-274.png)

   Save and you're done!

</Stepper>

You can also highlight the text you’d like to link, then click on the Link icon in the menu and select **File manager**. This will take you directly to your Library, where you can either add a new file or choose an existing one to link to the selected text.



## Troubleshooting

**PDF upload failure**

- Check that the file size is under the maximum limit (typically 10 MB)
- Ensure the PDF is not corrupted or password protected
- Try converting to PDF/A format for better compatibility

**Link doesn't work in email**

- Verify that the PDF was successfully uploaded to the file manager
- Check that the link URL is complete and properly formatted
- Test the link in preview mode before sending

**File manager storage full**

- Delete old or unused files from the file manager
- Compress PDFs before uploading to save space
- Consider upgrading your plan for more storage capacity
