# Uploading to the File Manager

## Description

Your library allows you to upload files such as images or PDFs. For example, you can store images that go with your established brand and structure. You can use this in several ways throughout your email campaigns.

## Why It's Important
Having all your images and files in one place will help you prepare to send your campaigns and enhance them with excellent visuals or useful content.

## Step-by-Step Instructions

The procedure for adding files to the file manager is as follows:

### For images in email content:
<Stepper>

1. Select **Edit Email** from the campaign bar
![Campaign edit button location](/images/campaign-creation/A5E92D9F-D74F-48BA-A833-017C5A80467E_4_5005_c.jpeg)
![Edit Email button in campaign bar](/images/campaign-creation/558DDDA4-0A8D-4A95-8086-0D375ABAA3FB_4_5005_c.jpeg)

1. In the Email Preview section, select **Edit Email**
![Edit Email button in preview section](/images/campaign-creation/C70DF271-09A8-4909-834F-2D307160106F_4_5005_c.jpeg)

1. In the Content menu, select **IMAGE** and drag the button to the email content
![Drag image content block to email](/images/campaign-creation/8F54E671-7BCF-42E9-A4B2-608DB413E3A8.png)

1. Click on **Browse**
![Browse button for image selection](/images/campaign-creation/04BF58E7-BBE8-450A-85CF-FB37F960B0D0_4_5005_c.jpeg)

1. You now have three options to add files:
   - **Upload** - Upload your own files
   - **Import** - Import from external sources
   - **Browse** - Browse from the free photo selection.
   ![File upload options interface](/images/campaign-creation/D18A765E-5CA0-4C5A-9117-48B46310CB81_4_5005_c.jpeg)

1. Click on **Upload** then select your file

1. Your file is now in our file manager and can be selected from there
**Note:** Currently, you can only upload one file at a time (no FTP is available)
</Stepper>

### For file links:
<Stepper>

1. Click on the **link button** in the section where you want to include your file.

1. Use the URL type dropdown, choose **File Manager**

1. Select the file you uploaded earlier from your library.

1. The file is now linked inside the email.

1. Recipients can access it by clicking on the link
</Stepper>

## File Limitations
- **Maximum image size**: 1600px by 2000px
- **Maximum file size**: 1MB

## Troubleshooting

**Upload failure error**

- Check that file size is under the 1MB limit
- Verify image dimensions don't exceed 1600x2000px
- Make sure file format is supported (JPG, PNG, GIF, PDF)

**File doesn't appear in library**

- Refresh the browser page after upload
- Clear browser cache if files are missing
- Check available storage space in your account

**Image quality issues**

- Optimize images before uploading (72-150 DPI for web)
- Use JPG for photos, PNG for graphics with transparency
- Compress large files without losing visible quality

**Can't select uploaded file**

- Wait for upload to complete fully
- Check file permissions and ownership
- Try re-uploading if the file seems corrupted

**Storage limit reached**

- Delete unused files from the library
- Compress files before uploading
- Contact support to increase storage capacity

**Slow upload speed**

- Check internet connection stability
- Upload during off-peak hours
- Break large uploads into smaller batches
