# Using the Email Designer

## Description

This is the main editing tool you'll use to create an email template; it's easy to use and allows for quick addition and removal of sections, setting different colors for different areas, deciding on the font and width of your campaign, testing and much more, all from a single window.

## Why It's Useful

This editing function allows users to create engaging email campaigns without writing code to accomplish any desired goal. With this tool, anyone can make an excellent email. The Email Designer is now more flexible with section moving, content reorganization, color picker functionality, and content versioning. Here we'll explore everything about our designer.

## How to Use the Email Designer

Here are some features of this tool to get started and use it to its full potential.

### Settings

This part is for setting the tone for the entire campaign. There are many options to consider:

- Content area width (we recommend not exceeding 600 pixels)

- Content area alignment. (Left or Center)

- Background color

- Content area background color

- Default font

- Link color

![Email designer settings panel showing background colors, fonts, and layout options](/images/campaign-creation/campaigns-using-the-email-designer-312.png)

#### Add your preheader

To get the most out of your campaign, don't forget to add a compelling preheader. Simply scroll down in your settings to your Optional Properties, and enter it in the preheader field. It's that simple!

![Preheader text field in optional properties section](/images/campaign-creation/campaigns-using-the-email-designer-313.png)

If you need some inspiration, our AI feature is there to help you generate engaging preheaders that resonate with your audience. Give it a try and find the perfect fit for your message.

![AI-powered preheader generation interface](/images/campaign-creation/campaigns-using-the-email-designer-314.png)

### Rows

Before adding content to your campaign, you need to choose a row. The Rows section consists of a variety of component structures. Please select the one you want and drag it anywhere in the campaign. Test them and find the one that works best for your template.

![Row layout options showing various column configurations](/images/campaign-creation/campaigns-using-the-email-designer-315.png)

![Dragging and dropping a row layout into the email canvas](/images/campaign-creation/campaigns-using-the-email-designer-316.png)

#### Move / Delete / Duplicate row

Reposition, delete, or duplicate a section of the email:

- First, select the row you need to reposition to access its repositioning menu.

To move a section up or down in the template:

- In the positioning menu, click on the crossed arrows found on the left to move your selected row.

![Row positioning controls with crossed arrows to move sections](/images/campaign-creation/campaigns-using-the-email-designer-317.png)

To delete a section:

- In the positioning menu, click on the trash icon on the right to delete the row.

![Row deletion button with trash icon](/images/campaign-creation/campaigns-using-the-email-designer-318.png)

To duplicate the row:

- In the positioning menu, click on the double window button on the right, next to the trash icon.

![Row duplication button with double window icon](/images/campaign-creation/campaigns-using-the-email-designer-319.png)

#### Edit

Several elements can be modified in the row section. Select the row you want to modify to access this menu on the right side of the designer.

Here are the features you can control:

- Row background color

- Content background color

- Don't stack on mobile

- Row background image

- Customize column

- Display condition

- Hide on Desktop/Mobile

- Customize columns

- Column background

- Padding

- Borders

![Row customization panel showing column, padding, and border settings](/images/campaign-creation/campaigns-using-the-email-designer-320.png)

### Content

Having decided what type of rows you want, you can now choose what content to include in those rows.

The following content options are available:

![Content block palette showing all available elements like Title, Paragraph, Image, Button, etc.](/images/campaign-creation/campaigns-using-the-email-designer-321.png)

- **Title**: Use this to insert your campaign title.

- **Paragraph**: Use this to insert a paragraph of text.

- **List**: Use this to insert a list of different content.

- **Image**: Use this to insert an image you've uploaded or a stock image.

- **Button**: Use this to insert a visible call-to-action button.

- **Divider**: Use this to divide your content.

- **Spacer**: This block can insert space into your email content.

- **Social**: Insert social media icons and link them to your pages.

- **HTML**: Insert your custom HTML code into the editor with this option.

- **Video**: Insert a thumbnail showing a preview of your video and a link to it.

- **Menu**: Use this to include an interactive menu in your email content.

- **GIFS**: Use this to include an animated gif in your email and choose from various options.

Each of these is editable within its block. Drag and drop the desired element into your email content section to use them. To create the perfect campaign, experiment with all the options in the designer. There's almost no limit to what this tool can do!

#### Revert to a previous version of a campaign

A previous version of your email can be restored. In the designer, click on the "clock" icon at the bottom left of your campaign. You can then revert to a specific action in editing.

![Version history clock icon in bottom left corner](/images/campaign-creation/campaigns-using-the-email-designer-322.png)

![Version history panel showing list of saved versions with timestamps](/images/campaign-creation/campaigns-using-the-email-designer-323.png)

#### Preview mode

You can preview the site in desktop and mobile mode by clicking on the Eye icon button found at the top right of the designer. Test your content on both platforms and make sure it looks good.

![Preview mode button with eye icon for desktop and mobile views](/images/campaign-creation/campaigns-using-the-email-designer-324.png)

#### Save changes to the campaign.

![Green save button to preserve campaign changes](/images/campaign-creation/campaigns-using-the-email-designer-325.png)

Always click the green Save button when modifying your campaign in the email designer. Better safe than sorry!

#### Save campaign as template, reset it, or abandon it

If you want to choose one of these options, click on the arrow button at the top right of the screen. You can learn more about saving templates here.

![Additional options dropdown menu with Save as Template, Reset, and abandon options](/images/campaign-creation/campaigns-using-the-email-designer-326.png)

#### Campaign details

If you're satisfied with your template and want to send it, click Continue in the upper right corner of the designer. The Campaign Details section provides you with these options to send the campaign.

You can set the following here:

- Your campaign name

- The subscriber list

- The segment in that audience,

- The sender name

- The sender email address

- The subject line

You can also review the campaign with a test email and schedule delivery by selecting Send Date and Time.

![Campaign details form showing fields for name, list, segment, sender info, and subject line](/images/campaign-creation/campaigns-using-the-email-designer-327.png)

## Troubleshooting

**Email designer won't load**

- Clear browser cache and cookies
- Use Chrome or Firefox for best compatibility
- Disable ad blockers and browser extensions

**Changes won't save**

- Click the green Save button frequently
- Check internet connection stability
- Look for auto-save confirmation messages

**Mobile preview issues**

- Use real device testing for accuracy
- Check "Don't stack on mobile" settings
- Adjust column widths for mobile display

**Images won't insert**

- Upload images to the library first
- Check image file size (max 1MB)
- Verify image dimensions (max 1600x2000px)

**Custom HTML not working**

- Validate HTML code before inserting
- Remove JavaScript (not supported in emails)
- Use inline CSS instead of external stylesheets

**Version history issues**

- Version history is limited to recent changes
- Save important versions as templates
- Cannot recover versions older than 30 days
