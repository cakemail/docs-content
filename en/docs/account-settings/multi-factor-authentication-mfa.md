# Multi-Factor Authentication (MFA)

## Description

Multi-factor authentication (MFA) is a security measure that strengthens the protection of your account. By enabling MFA, you actively improve your account's defense against unauthorized access. This guide provides step-by-step instructions for enabling MFA on your account.

## Why it matters

In today's digital world, protecting your online accounts is essential. Multi-factor authentication (MFA) provides an additional layer of security, reducing the risk of unauthorized access and potential data breaches. Implementing MFA is a proactive step toward strengthening your account security.

## Step-by-step instructions

<Stepper>

1. Click on your company name/email address in the top right corner (this opens a dropdown menu), then click Users

1. Choose the user for MFA activation

1. Scroll down to "Security" and click "Enable" to activate 2-step verification
![Enable 2-step verification](/images/accounts/accounts-setting-up-multi-factor-authentication-mfa-for-your-account-155.png)
</Stepper>

## Registration with authenticator app

Users can register using their authenticator app by scanning the QR code with their mobile device.

![Scan QR code with authenticator app](/images/accounts/accounts-setting-up-multi-factor-authentication-mfa-for-your-account-156.png)

Enter the 6-digit code displayed in your authenticator app to complete the MFA setup.

![Enter 6-digit verification code](/images/accounts/accounts-setting-up-multi-factor-authentication-mfa-for-your-account-157.png)

## Backup codes for additional security

If you lose access to your mobile device, you can use one of the 10 backup security codes. These codes serve as backup security, granting access when your primary device is unavailable. Store these codes securely.

## Independence to customize

With multi-factor authentication (MFA), each account user can independently enable or disable MFA according to their preferences and needs.

## Troubleshooting

**Cannot scan QR code**
- Ensure camera permissions are enabled
- Increase screen brightness
- Use manual entry option if available
- Try a different authenticator app

**6-digit code not working**
- Check that your device time is synchronized
- Enter the code quickly (expires in 30 seconds)
- Make sure you're using the correct account in the app
- Try generating a new code

**Lost access to authenticator app**
- Use one of your 10 backup codes
- Contact support with account verification
- Have account recovery information ready
- Consider storing backup codes securely

**Backup codes not working**
- Make sure you're entering the code correctly
- Each backup code can only be used once
- Check if the code has already been used
- Contact support if all codes fail

**Want to disable MFA**
- Go to security settings in your profile
- Click "Disable" next to 2-step verification
- Confirm with your password
- Consider security implications before disabling
