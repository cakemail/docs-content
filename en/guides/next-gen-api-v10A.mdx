---
title: "Python and PHP Next-gen API clients"
---

Introducing the Python and PHP Next-gen API clients, enabling developers to integrate the Cakemail Next-gen APIs more easily and quickly.

- The Python client [is available on PyPI](https://pypi.org/project/cakemail/)
- The PHP client [is available on Packagist](https://packagist.org/packages/cakemail/)

The [Integration guides](https://cakemail.dev/docs) and [API reference](https://cakemail.dev/reference) now include sample code snippets for both languages.

The languages of our future API clients are prioritized according to the developers' demand. Tell us about your projects!
