title: "Next-gen API v1.13"
---

Introducing multiple segments selection in campaigns and extending events support.

## What’s new

- We're now allowing multi-attachment support for transactional emails, enabling you to send various file types as png, jpeg, jpg.
- You can define[Interests](https://cakemail.dev/reference/listinterests) to classify and manage your subscribers.
- We have implemented Azure AD authentication for Single Sign-On (SSO) to access the Cakemail API. This feature is available on request for Enterprise accounts only.
- Our library has been updated to ensure enhanced security.

## Bug fixes

- For users experiencing discrepancies in the display of their physical address information, this issue has been resolved by using more detailed information.
