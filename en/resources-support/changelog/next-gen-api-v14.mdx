title: "Next-gen API v1.4"
---

600 Campaign Blueprints and new search functionalities in this release

- Blueprints! You now have access to more than 600 Campaign Blueprints to use as a starting point for your next email campaign.  
  Access [API documentation for Campaign Blueprints](https://cakemail.dev/v1.4.0/reference#campaign-blueprint)

Plus, a few more features and changes:

- Apply tags to Templates for easy categorization [API documentation for Campaign Blueprints](https://cakemail.dev/v1.4.0/reference#listcampaignblueprints)
- Search Templates by name and tags  [API documentation for Templates](https://cakemail.dev/v1.4.0/reference#template)
- Search Transaction Email logs by email [API documentation for logs Transactional Email](https://cakemail.dev/v1.4.0/reference#log-1)
- Usage of "-,\_" characters in Custom Attributes name is now allowed [API documentation for Account Custom Attributes](https://cakemail.dev/v1.4.0/reference#custom-attribute)  
  Finally, some minor bug fixes and invisible changes:
- Rate limiting on key endpoints, protecting the API and the sending reputation of your account.  On infrequent circumstances, the API can now return HTTP 429.
