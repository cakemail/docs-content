---
title: "Next-gen API v1.11"
---

Introducing multiple segments selection in campaigns and extending events support.

## What’s new

- Webhook on Email.Sent events now support automation emails and transactional emails.
- You can now send email campaigns to multiple segments at once.
- System emails now have a Sender language parameter.

## Bug fixes

- There was an issue with the Contact filters that prevented all of the data from being exported. The contact filtering functionality has been disabled for the time being while we work on improving it.
- In some cases, emails without links were not sent. This issue has been resolved.
- We fixed the Sender name personalization tag in the system emails.
