## Improvements to Webhook Events and Email API Functionality

We’ve enhanced the Webhooks for User.Confirmed, User.Deleted, and User.Created events, expanding their capabilities and fixing several bugs. We also improved tag filtering and resolved issues related to rendering. We made improvements to email tracking for more accurate insights.

For the Email API, we’ve added support for direct thumbnail rendering.
