## 🎨 Principios de diseño

### 1. **Enfoque móvil primero**
- Más del 50% de los correos se abren en móvil
- Los diseños de una sola columna funcionan mejor
- Tamaño de fuente mínimo de 14px
- Objetivos táctiles mínimos de 44px

### 2. **Jerarquía visual**
- Estructura clara de titulares
- Bloques de contenido escaneables
- Uso estratégico del espacio en blanco
- Una llamada a la acción principal

### 3. **Accesibilidad**
- Texto alternativo para todas las imágenes
- Contraste de color suficiente (4.5:1)
- Orden de lectura lógico
- Compatibilidad con lectores de pantalla

## 💡 Directrices esenciales de diseño

### Mejores prácticas de diseño
- **Ancho**: Máximo 600-640px
- **Pre-encabezado**: 40-90 caracteres
- **Parte visible inicial**: Mensaje clave y CTA
- **Estructura**: Pirámide invertida (importante → detalles)

### Tipografía
- **Fuentes**: Fuentes seguras para web (Arial, Georgia, Times)
- **Tamaños**: 14-16px cuerpo, 22-28px titulares
- **Altura de línea**: 1.5x para legibilidad
- **Colores**: Alto contraste para legibilidad

### Imágenes
- **Formato**: JPEG para fotos, PNG para gráficos
- **Tamaño**: Optimizar para web (menos de 1MB total)
- **Proporción**: 60% texto, 40% imágenes
- **Visualización**: Siempre incluir texto alternativo

## 📊 Métricas de rendimiento

Los correos electrónicos bien diseñados logran:
- **Tasas de apertura más altas**: Vista previa atractiva
- **Mejores tasas de clics**: CTAs claros
- **Menos cancelaciones de suscripción**: Apariencia profesional
- **Mejores conversiones**: Diseño amigable para el usuario

## 🚀 Proceso de diseño

### 1. **Planificar** (10 minutos)
- Definir objetivo del correo
- Identificar mensaje clave
- Elegir plantilla

### 2. **Diseñar** (30 minutos)
- Crear jerarquía visual
- Escribir texto atractivo
- Agregar imágenes estratégicamente

### 3. **Probar** (20 minutos)
- Vista previa en múltiples dispositivos
- Verificar renderizado en clientes de correo
- Verificar que todos los enlaces funcionen

### 4. **Optimizar** (continuo)
- Pruebas A/B de elementos de diseño
- Analizar datos de rendimiento
- Iterar basándose en resultados

## ⚠️ Errores comunes de diseño

- Usar demasiadas fuentes o colores
- Descuidar la optimización móvil
- Crear correos electrónicos solo de imágenes
- Ocultar texto importante en imágenes
- Olvidar el texto alternativo
- Contraste de color pobre
- Diseños desordenados

## 🛠️ Herramientas y recursos de diseño

### Pruebas de correo electrónico
- Vista previa en múltiples clientes
- Verificar puntuaciones de spam
- Verificar renderizado

### Optimización de imágenes
- Comprimir sin pérdida de calidad
- Usar formatos apropiados
- Implementar imágenes responsivas

### Verificación de accesibilidad
- Analizadores de contraste de color
- Pruebas con lectores de pantalla
- Verificación de texto alternativo
