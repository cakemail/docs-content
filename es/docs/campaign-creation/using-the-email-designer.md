# Usar el diseñador de correo electrónico

<iframe src="//www.youtube-nocookie.com/embed/az96wcaWOnA" width="560" height="315" frameborder="0" allowfullscreen=""></iframe>

## Descripción

Esta es la herramienta principal de edición que usarás para crear una plantilla de correo electrónico; es fácil de usar y permite agregar y eliminar secciones rápidamente, establecer diferentes colores para diferentes áreas, decidir la fuente y el ancho de tu campaña, probar y mucho más, todo desde una sola ventana.

## Por qué es útil

Esta función de edición permite a los usuarios crear campañas de correo electrónico atractivas sin escribir código para lograr cualquier objetivo deseado. Con esta herramienta, cualquiera puede hacer un excelente correo electrónico. El diseñador de correo electrónico ahora es más flexible con el movimiento de secciones, reorganización de contenido, funcionalidad de selector de color y versionado de contenido. Aquí exploraremos todo sobre nuestro diseñador.

## Cómo usar el diseñador de correo electrónico

Aquí hay algunas características de esta herramienta para comenzar y usarla a su máximo potencial.

### Configuración

Esta parte es para establecer el tono de toda la campaña. Hay muchas opciones a considerar:

- Ancho del área de contenido (recomendamos no exceder 600 píxeles)

- Alineación del área de contenido. (Izquierda o Centro)

- Color de fondo

- Color de fondo del área de contenido

- Fuente predeterminada

- Color de enlaces

![Email designer settings panel showing background colors, fonts, and layout options](/images/campaign-creation/campaigns-using-the-email-designer-312.png)

#### Agregar tu preencabezado

Para aprovechar al máximo tu campaña, no olvides agregar un preencabezado convincente. Simplemente desplázate hacia abajo en tu configuración hasta Propiedades Opcionales, e ingrésalo en el campo de preencabezado. ¡Es así de simple!

![Preheader text field in optional properties section](/images/campaign-creation/campaigns-using-the-email-designer-313.png)

Si necesitas algo de inspiración, nuestra función de IA está ahí para ayudarte a generar preencabezados atractivos que resuenen con tu audiencia. Pruébala y encuentra el ajuste perfecto para tu mensaje.

![AI-powered preheader generation interface](/images/campaign-creation/campaigns-using-the-email-designer-314.png)

### Filas

Antes de agregar contenido a tu campaña, necesitas elegir una fila. La sección Filas consiste en una variedad de estructuras de componentes. Por favor selecciona la que quieras y arrástrala a cualquier lugar de la campaña. Pruébalas y encuentra la que funcione mejor para tu plantilla.

![Row layout options showing various column configurations](/images/campaign-creation/campaigns-using-the-email-designer-315.png)

![Dragging and dropping a row layout into the email canvas](/images/campaign-creation/campaigns-using-the-email-designer-316.png)

#### Mover / Eliminar / Duplicar fila

Reposicionar, eliminar o duplicar una sección del correo electrónico:

- Primero, selecciona la fila que necesitas reposicionar para acceder a su menú de reposicionamiento.

Para mover una sección hacia arriba o abajo en la plantilla:

- En el menú de posicionamiento, haz clic en las flechas cruzadas que se encuentran a la izquierda para mover tu fila seleccionada.

![Row positioning controls with crossed arrows to move sections](/images/campaign-creation/campaigns-using-the-email-designer-317.png)

Para eliminar una sección:

- En el menú de posicionamiento, haz clic en el ícono de basura a la derecha para eliminar la fila.

![Row deletion button with trash icon](/images/campaign-creation/campaigns-using-the-email-designer-318.png)

Para duplicar la fila:

- En el menú de posicionamiento, haz clic en el botón de doble ventana a la derecha, junto al ícono de basura.

![Row duplication button with double window icon](/images/campaign-creation/campaigns-using-the-email-designer-319.png)

#### Editar

Varios elementos pueden ser modificados en la sección de fila. Selecciona la fila que quieres modificar para acceder a este menú en el lado derecho del diseñador.

Aquí están las características que puedes controlar:

- Color de fondo de la fila

- Color de fondo del contenido

- No apilar en móvil

- Imagen de fondo de la fila

- Personalizar columna

- Condición de visualización

- Ocultar en Escritorio/Móvil

- Personalizar columnas

- Fondo de columna

- Relleno

- Bordes

![Row customization panel showing column, padding, and border settings](/images/campaign-creation/campaigns-using-the-email-designer-320.png)

### Contenido

Habiendo decidido qué tipo de filas quieres, ahora puedes elegir qué contenido incluir en esas filas.

Las siguientes opciones de contenido están disponibles:

![Content block palette showing all available elements like Title, Paragraph, Image, Button, etc.](/images/campaign-creation/campaigns-using-the-email-designer-321.png)

- **Título**: Usa esto para insertar el título de tu campaña.

- **Párrafo**: Usa esto para insertar un párrafo de texto.

- **Lista**: Usa esto para insertar una lista de contenido diferente.

- **Imagen**: Usa esto para insertar una imagen que hayas subido o una imagen de archivo.

- **Botón**: Usa esto para insertar un botón de llamada a la acción visible.

- **Divisor**: Usa esto para dividir tu contenido.

- **Espaciador**: Este bloque puede insertar espacio en tu contenido de correo electrónico.

- **Social**: Inserta íconos de redes sociales y enlázalos a tus páginas.

- **HTML**: Inserta tu código HTML personalizado en el editor con esta opción.

- **Video**: Inserta una miniatura mostrando una vista previa de tu video y un enlace a él.

- **Menú**: Usa esto para incluir un menú interactivo en tu contenido de correo electrónico.

- **GIFS**: Usa esto para incluir un gif animado en tu correo electrónico y elegir entre varias opciones.

Cada uno de estos es editable dentro de su bloque. Arrastra y suelta el elemento deseado en tu sección de contenido de correo electrónico para usarlos. Para crear la campaña perfecta, experimenta con todas las opciones en el diseñador. ¡Casi no hay límite a lo que esta herramienta puede hacer!

#### Revertir a una versión anterior de una campaña

Se puede restaurar una versión anterior de tu correo electrónico. En el diseñador, haz clic en el ícono de "reloj" en la parte inferior izquierda de tu campaña. Luego puedes revertir a una acción específica en la edición.

![Version history clock icon in bottom left corner](/images/campaign-creation/campaigns-using-the-email-designer-322.png)

![Version history panel showing list of saved versions with timestamps](/images/campaign-creation/campaigns-using-the-email-designer-323.png)

#### Modo de vista previa

Puedes previsualizar el sitio en modo escritorio y móvil haciendo clic en el botón del ícono del Ojo que se encuentra en la parte superior derecha del diseñador. Prueba tu contenido en ambas plataformas y asegúrate de que se vea bien.

![Preview mode button with eye icon for desktop and mobile views](/images/campaign-creation/campaigns-using-the-email-designer-324.png)

#### Guardar cambios en la campaña.

![Green save button to preserve campaign changes](/images/campaign-creation/campaigns-using-the-email-designer-325.png)

Siempre haz clic en el botón verde Guardar cuando modifiques tu campaña en el diseñador de correo electrónico. ¡Más vale prevenir que lamentar!

#### Guardar campaña como plantilla, restablecerla o abandonarla

Si quieres elegir una de estas opciones, haz clic en el botón de flecha en la parte superior derecha de la pantalla. Puedes aprender más sobre guardar plantillas aquí.

![Additional options dropdown menu with Save as Template, Reset, and abandon options](/images/campaign-creation/campaigns-using-the-email-designer-326.png)

#### Detalles de la campaña

Si estás satisfecho con tu plantilla y quieres enviarla, haz clic en Continuar en la esquina superior derecha del diseñador. La sección Detalles de la Campaña te proporciona estas opciones para enviar la campaña.

Puedes establecer lo siguiente aquí:

- El nombre de tu campaña

- La lista de suscriptores

- El segmento en esa audiencia,

- El nombre del remitente

- La dirección de correo electrónico del remitente

- La línea de asunto

También puedes revisar la campaña con un correo electrónico de prueba y programar la entrega seleccionando Fecha y Hora de Envío.

![Campaign details form showing fields for name, list, segment, sender info, and subject line](/images/campaign-creation/campaigns-using-the-email-designer-327.png)

Recuerda probarlo completamente antes del envío por correo electrónico. Aquí hay una lista de verificación de cosas que hacer antes de enviar una campaña y después de que haya sido enviada.

## Solución de problemas

**El diseñador de correo electrónico no carga**

- Borra la caché del navegador y las cookies
- Usa Chrome o Firefox para mejor compatibilidad
- Desactiva bloqueadores de anuncios y extensiones del navegador

**Los cambios no se guardan**

- Haz clic en el botón verde Guardar frecuentemente
- Verifica la estabilidad de la conexión a internet
- Busca mensajes de confirmación de guardado automático

**Problemas de vista previa móvil**

- Usa pruebas en dispositivos reales para mayor precisión
- Verifica la configuración "No apilar en móvil"
- Ajusta el ancho de las columnas para pantalla móvil

**Las imágenes no se insertan**

- Sube imágenes a la biblioteca primero
- Verifica el tamaño del archivo de imagen (máx. 1MB)
- Verifica las dimensiones de la imagen (máx. 1600x2000px)

**El HTML personalizado no funciona**

- Valida el código HTML antes de insertarlo
- Elimina JavaScript (no soportado en correos electrónicos)
- Usa CSS en línea en lugar de hojas de estilo externas

**Problemas de historial de versiones**

- El historial de versiones está limitado a cambios recientes
- Guarda versiones importantes como plantillas
- No se pueden recuperar versiones de más de 30 días
