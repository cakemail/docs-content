# Dirección de remitente e idioma predeterminado

## Descripción

La configuración de dirección de remitente e idioma predeterminado controla cómo aparecen tus correos automatizados de bienvenida y confirmación a nuevos suscriptores. Esta función te permite personalizar las preferencias de idioma para todos los correos generados por el sistema, incluyendo confirmaciones de suscripción, mensajes de bienvenida y confirmaciones de cancelación de suscripción. La configuración adecuada asegura que estos puntos de contacto automatizados se alineen con tu identidad de marca y proporcionen una experiencia consistente a través de todas las interacciones del suscriptor.

## Por qué es importante

Para construir tu marca y ser reconocible por los destinatarios, debes elegir el idioma correcto para ser reconocido por tus destinatarios.

## Instrucciones paso a paso

![Establecer remitente e idioma predeterminado](/images/forms/Nov-09-2021_11-11-00.gif)

Sigue estos pasos simples para especificar adecuadamente el remitente predeterminado para tu lista y sus correos de confirmación automatizados:

<Stepper>

1. Selecciona la pestaña **Contactos**
![Seleccionar la pestaña Contactos](/images/forms/B7680B8E-0822-4DE1-A29F-A3EEB9FE5DB8_4_5005_c.jpeg)

1. Elige tu lista de contactos del menú desplegable
![Elegir lista de contactos](/images/forms/ACDED713-00D2-45DD-92E8-BA34863B7A27_4_5005_c.jpeg)

1. Haz clic en el **icono de engranaje** para acceder al menú de gestión de listas
![Hacer clic en el icono de engranaje](/images/forms/CD6FF683-4314-4B46-A481-B23A2F173071_4_5005_c.jpeg)

1. En el menú del gestor de listas, haz clic en la pestaña **Configuración**
![Hacer clic en la pestaña Configuración](/images/forms/BA70BCD2-FF6F-4263-9E74-5E1179E5F04F_4_5005_c.jpeg)

1. Selecciona tu remitente de la opción **Remitente predeterminado**
![Seleccionar remitente predeterminado](/images/forms/04506F34-5070-40C0-A476-FD33B38C9515_4_5005_c.jpeg)
Puedes agregar tu correo de remitente si no lo encuentras en la lista seleccionando **Gestionar remitentes**.
*Referencia: Agregar un nuevo remitente*

1. Finalmente, puedes establecer el idioma predeterminado para la lista desde este menú. Selecciona la pestaña **Configuración** para establecer el **Idioma predeterminado** para la lista
![Establecer idioma predeterminado](/images/forms/882BB14B-BD5B-4ACF-BACD-CB920E8E686A.png)
Este es el idioma predeterminado que los usuarios verán al gestionar sus suscripciones. Esto no afectará el idioma de tus campañas de correo.
</Stepper>

## Solución de problemas

**No puedo encontrar mi remitente en la lista desplegable**

- Haz clic en "Gestionar remitentes" para agregar un nuevo correo de remitente
- Verifica que el correo del remitente haya sido confirmado
- Verifica si tienes permisos para gestionar remitentes
- Asegúrate de que el remitente no esté archivado o eliminado

**La configuración de idioma no se aplica correctamente**

- Confirma que guardaste los cambios después de seleccionar el idioma
- Verifica si el idioma del navegador está anulando la configuración
- Confirma que el idioma seleccionado sea compatible
- Prueba con un nuevo suscriptor para ver la configuración actual

**Los correos de bienvenida muestran información incorrecta**

- Revisa que la configuración de remitente e idioma esté guardada
- Verifica si hay configuraciones conflictivas a nivel de cuenta
- Asegúrate de que el idioma de la plantilla coincida con el idioma predeterminado
- Confirma que los cambios se hayan propagado (puede tomar unos minutos)
