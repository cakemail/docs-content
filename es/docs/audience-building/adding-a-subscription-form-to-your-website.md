# Agregar un formulario de suscripción a tu sitio web

<iframe width="560" height="315" src="https://www.youtube-nocookie.com/embed/QEhJ3QzE_Xo" frameborder="0" 
 allowfullscreen></iframe>
 
## Descripción

Los formularios de suscripción se pueden incluir en tu sitio web o blog para que los visitantes puedan proporcionar sus datos para recibir correos sobre temas relevantes y/o interesantes. Básicamente, un formulario de suscripción está diseñado para registrar suscriptores a tu lista de correo.

## Por qué es importante

Los formularios de suscripción son una forma legal y conveniente para que los suscriptores te proporcionen su información. El proceso de creación y configuración es muy simple. Una de las grandes ventajas del marketing por correo es que te permite dirigirte a grupos basándote en varios criterios. Al agregar algunos de estos criterios a tu formulario de suscripción, puedes personalizar tus correos. Sin embargo, como regla general, pedir demasiada información puede llevar a menos interés en suscribirse. Necesitarás comenzar creando un formulario que esté adaptado a tus necesidades.

## Instrucciones paso a paso

Usando nuestro sistema, puedes integrar un formulario de opt-in para una lista de contactos específica directamente en tu sitio web.

### Obtener el formulario para tu lista

<Stepper>

1. Selecciona la pestaña **Formularios**
   ![Navegación de la pestaña Formularios](/images/forms/DE2647B1-84B2-48DC-8709-CB03D406C3CC_4_5005_c.jpeg)

1. Haz clic en **'...'** en la esquina superior derecha del formulario de suscripción, luego selecciona **Editar**
   ![Menú de opciones del formulario con botón Editar](/images/forms/6EE9D232-BC7A-419B-AD95-FC24288C49E0_4_5005_c.jpeg)

1. Tendrás la opción de:
   - usar un enlace directo al formulario
   - obtener el código HTML para incrustar en tu sitio
</Stepper>

### Formulario vinculado

Te dará el enlace directo al formulario haciendo clic en **Obtener URL**.

![Opción Obtener URL](/images/forms/994FE0B7-094B-41B1-8786-97C21DB033D8_4_5005_c.jpeg)

![Interfaz del editor de formularios](/images/forms/D4898F41-2133-4A78-9EDA-E86A0CA5F44F_4_5005_c.jpeg)

Puedes personalizar tu formulario haciendo clic en **Editar formulario**.

![Personalización de edición del formulario](/images/forms/89BDF6DB-831B-40A7-AEBA-2B1F77B4FDF1_4_5005_c.jpeg)

### Formulario incrustado

Te dará el código para incrustar en tu sitio web seleccionando **Obtener código**.

![Opción Obtener código](/images/forms/FE932224-C921-4065-84CB-582AADD2AB5C_4_5005_c.jpeg)

![Vista previa del formulario incrustado](/images/forms/FB295771-7AEB-4F17-9E75-B0C14F2AB255_4_5005_c.jpeg)

Siéntete libre de personalizar tu formulario haciendo clic en **Editar formulario**.

![Interfaz de personalización del formulario](/images/forms/87056EFD-3136-4C7E-9F20-88E977D2D298_4_5005_c.jpeg)

### Editar tu formulario

Los siguientes elementos de formulario están disponibles en tu editor:

- **Título** con seis tamaños H diferentes
- **Párrafo:** Describe el tipo de comunicación que se enviará. Comunicación semanal o mensual, por ejemplo. Códigos de registro, códigos promocionales, etc.
- **Atributo personalizado:** Un atributo que se puede personalizar
- **Correo electrónico** (requerido y por defecto)
- **Botón de suscripción** (requerido y por defecto)
- **Atributo oculto** en casos donde quieres que un segmento se pueble por defecto

![Personalización de elementos del formulario](/images/forms/50A54B79-A700-4E53-83CA-E43A179A3378_4_5005_c.jpeg)

Puedes seleccionar estos elementos para personalizarlos. Es importante evitar pedir la información incorrecta: no quieres que tus clientes lo encuentren abrumador, así que simplemente dejan de suscribirse. Tres o cuatro campos deberían ser suficientes.

![Opciones de elementos del formulario](/images/forms/1F845091-4D60-4758-A9BD-DE622A2E6E01_4_5005_c.jpeg)

- Haz clic en el **icono de lápiz** para editar un elemento
- Haz clic en el **cuadrado doble con signo más** para duplicar un elemento
- Haz clic en el **icono de basura** para eliminar un elemento

## Tu formulario de suscripción está listo

Úsalo para construir tu valiosa lista y continuar generando prospectos para el crecimiento de tu cuenta. Los datos que recojas proporcionarán una fuente de nuevos prospectos para tu organización, permitiéndole continuar expandiéndose. Además, si tienes campos personalizados en tu lista, se agregarán al formulario.

## Solución de problemas

**El formulario no se muestra en mi sitio web**

- Verifica que el código de incrustación se haya copiado completamente sin caracteres faltantes
- Verifica si la configuración de seguridad de tu sitio web está bloqueando scripts externos
- Asegúrate de que el código del formulario esté colocado dentro de las etiquetas body de tu HTML
- Prueba en diferentes navegadores para eliminar problemas específicos del navegador

**Las presentaciones no aparecen en mi lista**

- Confirma que el formulario esté conectado a la lista de contactos correcta
- Verifica si el doble opt-in está habilitado (los suscriptores deben confirmar)
- Confirma que tu lista no haya alcanzado su límite de suscriptores
- Busca presentaciones en estado pendiente o no confirmado

**El estilo del formulario no coincide con mi sitio web**

- Usa las opciones de personalización CSS en el editor de formularios
- Agrega CSS personalizado para anular los estilos por defecto
- Verifica si el CSS de tu sitio web entra en conflicto con los estilos del formulario
- Considera usar la opción de incrustación iframe para mejor aislamiento

**Mensajes de error al enviar el formulario**

- Asegúrate de que todos los campos requeridos estén llenos correctamente
- Verifica si la validación de correo está bloqueando ciertos formatos de correo
- Confirma que el formulario no haya sido modificado después de incrustarlo
- Prueba con diferentes direcciones de correo para identificar patrones
