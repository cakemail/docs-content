# Usar etiquetas para rastrear el engagement

## Descripción

Puedes crear segmentos basados en acciones etiquetando contactos. Cuando etiquetas un contacto, actualizas su perfil en tiempo real. Por ejemplo, si tus suscriptores hacen clic en ciertos enlaces de correo electrónico, puedes etiquetarlos con etiquetas específicas.

En esencia, tus contactos de correo electrónico pueden ser clasificados según tus necesidades de segmentación creando etiquetas. También puedes etiquetar suscriptores basado en otras acciones, como la forma en que interactúan con tus productos. Te mostraremos cómo crear y usar etiquetas en este artículo, así como cómo hacerlo para toda tu lista de contactos.

## Por qué es importante

Las etiquetas son una herramienta poderosa para la segmentación avanzada en el marketing por correo electrónico. Te permiten categorizar contactos basado en su comportamiento, preferencias e interacciones con tus campañas en tiempo real. Este nivel granular de organización te permite enviar campañas altamente dirigidas a grupos específicos, mejorando las tasas de engagement y conversiones. Las etiquetas también te ayudan a rastrear las etapas del viaje del cliente, automatizar flujos de trabajo basados en acciones del suscriptor y mantener una base de datos de contactos limpia y bien organizada que escale con tu negocio.

## Instrucciones paso a paso

Así es como crear y usar una etiqueta.

<Stepper>

1. Haz clic en tu nombre de usuario en la esquina superior derecha y selecciona **Biblioteca de etiquetas** del menú desplegable
![Access tag library from user menu](/images/manage-contacts/manage-contacts-creating-and-using-tags-38.png)

1. Ingresa el nombre de tu nueva etiqueta
![Enter new tag name in text field](/images/manage-contacts/manage-contacts-creating-and-using-tags-39.png)

1. Haz clic en **Crear 1 etiqueta**
![Click Create 1 tag button](/images/manage-contacts/manage-contacts-creating-and-using-tags-40.png)

1. ¡Tu etiqueta ya está creada! Puedes cerrar el menú
![Tag created confirmation screen](/images/manage-contacts/manage-contacts-creating-and-using-tags-41.png)
</Stepper>

## Cómo etiquetar un contacto

Para usar una etiqueta, sigue estos pasos:

<Stepper>

1. Selecciona la pestaña **Contactos**
![Navigate to Contacts tab](/images/manage-contacts/manage-contacts-creating-and-using-tags-42.png)

1. Selecciona tu lista de contactos
![Select your contact list from available lists](/images/manage-contacts/manage-contacts-creating-and-using-tags-43.png)

1. Marca el círculo junto a la dirección de correo electrónico
![Select contact by checking the box](/images/manage-contacts/manage-contacts-creating-and-using-tags-44.png)

1. La parte inferior de la página mostrará un nuevo menú. Selecciona **Agregar o eliminar etiquetas** del menú Acciones
![Select Add or remove tags from Actions menu](/images/manage-contacts/manage-contacts-creating-and-using-tags-45.png)

1. Elige la etiqueta que quieres usar y haz clic en **Etiquetar**. Alternativamente, puedes crear una etiqueta directamente en este menú
![Select tag and click Tag button](/images/manage-contacts/manage-contacts-creating-and-using-tags-46.png)

1. Cuando hayas seleccionado la etiqueta, haz clic en **Listo**
![Click Done button to confirm tag selection](/images/manage-contacts/manage-contacts-creating-and-using-tags-47.png)

1. ¡Es así de simple, tu contacto ya está etiquetado!
![Successfully tagged contact confirmation](/images/manage-contacts/manage-contacts-creating-and-using-tags-48.png)
</Stepper>

## Segmentar tus contactos al importar una lista

También es posible incluir una etiqueta al importar tu lista de contactos, segmentando así automáticamente la lista durante la importación.

![Contact import interface with tags](/images/manage-contacts/manage-contacts-creating-and-using-tags-49.png)

Si ya tienes un segmento basado en una etiqueta, puedes seleccionar ese segmento durante el proceso de importación, y cada contacto en la lista será agregado automáticamente al segmento.

También puedes crear una nueva etiqueta o agregar una etiqueta existente a todos tus contactos durante el paso de validación de importación de lista para uso futuro.

## Cómo importar etiquetas existentes
Si ya estás usando etiquetas en otro servicio de correo electrónico, puedes importar tu lista con etiquetas separadas por ",". Durante el proceso de importación, las etiquetas se crearán individualmente.

## Solución de problemas

**Las etiquetas no aparecen en la biblioteca de etiquetas**

- Actualiza la página después de crear etiquetas
- Verifica si hay nombres de etiquetas duplicados
- Asegúrate de que las etiquetas se guardaron correctamente
- Cierra sesión y vuelve a iniciar sesión para actualizar la vista

**No se pueden agregar etiquetas a contactos**

- Verifica que los contactos estén seleccionados (casilla marcada)
- Confirma que las etiquetas existan en la biblioteca primero
- Asegúrate de tener permiso para editar contactos
- Intenta etiquetar un contacto a la vez

**Las etiquetas no se importan con los contactos**

- Incluye etiquetas en una columna CSV dedicada
- Usa separación por comas para múltiples etiquetas
- Crea etiquetas en la biblioteca antes de importar
- Verifica que el mapeo de importación incluya la columna de etiquetas

**No se pueden crear nuevas etiquetas**

- Verifica si hay caracteres especiales en los nombres de las etiquetas
- Asegúrate de que los nombres de las etiquetas no sean demasiado largos
- Confirma que no hayas alcanzado los límites de etiquetas
- Intenta crear etiquetas directamente desde la biblioteca de etiquetas

**Las etiquetas no funcionan en segmentos**

- Permite tiempo para que las etiquetas se procesen después de agregarlas
- Asegúrate de que las etiquetas estén escritas correctamente en los segmentos
- Verifica que los contactos etiquetados estén en la lista correcta
- Actualiza el segmento después de etiquetar contactos

**El etiquetado masivo falla**

- Reduce el número de contactos seleccionados
- Verifica problemas de tiempo de espera con lotes grandes
- Asegúrate de que todos los contactos seleccionados estén activos
- Intenta etiquetar en grupos más pequeños de 500 o menos
