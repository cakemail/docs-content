# Actualizar información de contacto

## Descripción

Tu lista de contactos puede ser modificada o actualizada en cualquier momento. Puedes, por ejemplo, ingresar un atributo, como un cumpleaños, para un contacto para diferenciarlo y dirigirte a ese destinatario. También están disponibles opciones para cancelar suscripción, eliminar o agregar un contacto a una lista de supresión.

## Por qué es importante

Algunos contactos pueden necesitar ser eliminados manualmente de tu lista para mantener una lista limpia, mientras que otros simplemente pueden necesitar que su información de contacto sea cambiada. Los desafíos resultantes en encontrar información valiosa en la lista de contactos en constante evolución y crecimiento hacen que administrar la información de contacto sea esencial para la eficiencia organizacional.

## Solución de problemas

**No se puede encontrar el contacto específico**

- Verifica si el correo electrónico se escribió correctamente en la búsqueda
- Intenta buscar con una dirección de correo electrónico parcial
- Asegúrate de que el contacto esté en la lista seleccionada
- Verifica si el contacto podría tener la suscripción cancelada

**La opción de editar no aparece**

- Asegúrate de haber hecho clic en el menú de tres puntos (...)
- Verifica que tengas permiso para editar contactos
- Confirma que el contacto no esté siendo procesado
- Intenta actualizar la página

**Los cambios del contacto no se guardan**

- Verifica si hay errores de validación en los campos
- Asegúrate de que el formato del correo electrónico sea válido
- Busca campos requeridos que estén vacíos
- Intenta editar un campo a la vez

**No se pueden agregar atributos al contacto**

- Verifica que los atributos existan para esta lista
- Confirma que el tipo de datos del atributo coincida con la entrada
- Asegúrate de que el formato de fecha sea YYYY-MM-DD
- Elimina cualquier espacio al final en los campos de texto

**Las acciones masivas no funcionan**

- Limita la selección a 500 contactos a la vez
- Asegúrate de que todos los contactos seleccionados estén activos
- Verifica que tengas los permisos necesarios
- Espera a que se complete la acción masiva anterior

**El contacto muestra un estado incorrecto**

- Permite tiempo para que se actualicen los cambios de estado
- Verifica el historial del contacto para cambios recientes
- Confirma que el contacto no esté en la lista de supresión
- Revisa los registros de cancelación de suscripción para este contacto
