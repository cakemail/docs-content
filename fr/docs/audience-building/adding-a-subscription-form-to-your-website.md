# Ajouter un formulaire d'abonnement à votre site web

<iframe width="560" height="315" src="https://www.youtube-nocookie.com/embed/QEhJ3QzE_Xo" frameborder="0" 
 allowfullscreen></iframe>
 
## Description

Les formulaires d'abonnement peuvent être inclus dans votre site web ou blog afin que les visiteurs puissent fournir leurs données pour recevoir des emails sur des sujets pertinents et/ou d'intérêt. Fondamentalement, un formulaire d'abonnement est conçu pour enregistrer des abonnés à votre liste de diffusion.

## Pourquoi c'est important

Les formulaires d'abonnement sont un moyen légal et pratique pour les abonnés de vous fournir leurs informations. Le processus de création et de configuration est très simple. Un des grands avantages du marketing par email est qu'il vous permet de cibler des groupes basés sur divers critères. En ajoutant certains de ces critères à votre formulaire d'abonnement, vous pouvez personnaliser vos emails. Comme règle générale cependant, demander trop d'informations peut mener à moins d'intérêt pour s'abonner. Vous devrez commencer par créer un formulaire qui est adapté à vos besoins.

## Instructions étape par étape

En utilisant notre système, vous pouvez intégrer un formulaire d'acceptation pour une liste de contacts spécifique directement dans votre site web.

### Obtenir le formulaire pour votre liste

<Stepper>

1. Sélectionnez l'onglet **Formulaires**
   ![Navigation de l'onglet Formulaires](/images/forms/DE2647B1-84B2-48DC-8709-CB03D406C3CC_4_5005_c.jpeg)

1. Cliquez sur **'...'** en haut à droite du formulaire d'abonnement puis sélectionnez **Modifier**
   ![Menu d'options de formulaire avec bouton Modifier](/images/forms/6EE9D232-BC7A-419B-AD95-FC24288C49E0_4_5005_c.jpeg)

1. Vous aurez l'option de :
   - utiliser un lien direct vers le formulaire
   - récupérer le code HTML pour intégrer sur votre site
</Stepper>

### Formulaire lié

Il vous donnera le lien direct vers le formulaire en cliquant sur **Obtenir l'URL**.

![Option Obtenir l'URL](/images/forms/994FE0B7-094B-41B1-8786-97C21DB033D8_4_5005_c.jpeg)

![Interface de l'éditeur de formulaire](/images/forms/D4898F41-2133-4A78-9EDA-E86A0CA5F44F_4_5005_c.jpeg)

Vous pouvez personnaliser votre formulaire en cliquant sur **Modifier le formulaire**.

![Personnalisation de modification de formulaire](/images/forms/89BDF6DB-831B-40A7-AEBA-2B1F77B4FDF1_4_5005_c.jpeg)

### Formulaire intégré

Il vous donnera le code à intégrer sur votre site web en sélectionnant **Obtenir le code**.

![Option Obtenir le code](/images/forms/FE932224-C921-4065-84CB-582AADD2AB5C_4_5005_c.jpeg)

![Aperçu du formulaire intégré](/images/forms/FB295771-7AEB-4F17-9E75-B0C14F2AB255_4_5005_c.jpeg)

N'hésitez pas à personnaliser votre formulaire en cliquant sur **Modifier le formulaire**.

![Interface de personnalisation de formulaire](/images/forms/87056EFD-3136-4C7E-9F20-88E977D2D298_4_5005_c.jpeg)

### Modifier votre formulaire

Les éléments de formulaire suivants sont disponibles dans votre éditeur :

- **Titre** avec six tailles H différentes
- **Paragraphe :** Décrivez le type de communication qui sera envoyé. Communication sur une base hebdomadaire ou mensuelle, par exemple. Codes d'inscription, codes promo, etc.
- **Attribut personnalisé :** Un attribut qui peut être personnalisé
- **Email** (obligatoire et par défaut)
- **Bouton d'abonnement** (obligatoire et par défaut)
- **Attribut caché** dans les cas où vous voulez qu'un segment soit populé par défaut

![Personnalisation d'élément de formulaire](/images/forms/50A54B79-A700-4E53-83CA-E43A179A3378_4_5005_c.jpeg)

Vous pouvez sélectionner ces éléments pour les personnaliser. Il est important d'éviter de demander les mauvaises informations : vous ne voulez pas que vos clients trouvent cela accablant, alors ils cessent simplement de s'abonner. Trois ou quatre champs devraient suffire.

![Options d'élément de formulaire](/images/forms/1F845091-4D60-4758-A9BD-DE622A2E6E01_4_5005_c.jpeg)

- Cliquez sur l'**icône de crayon** pour modifier un élément
- Cliquez sur le **double carré avec un signe plus** pour dupliquer un élément
- Cliquez sur l'**icône de poubelle** pour supprimer un élément

## Votre formulaire d'abonnement est prêt

Utilisez-le pour construire votre précieuse liste et continuez à générer des prospects pour la croissance de votre compte. Les données que vous assemblez fourniront une source de nouveaux prospects pour votre organisation, lui permettant de continuer à s'étendre. De plus, si vous avez des champs personnalisés sur votre liste, ils seront ajoutés au formulaire.

## Dépannage

**Le formulaire ne s'affiche pas sur mon site web**

- Vérifiez que le code d'intégration a été copié complètement sans caractères manquants
- Vérifiez si les paramètres de sécurité de votre site web bloquent les scripts externes
- Assurez-vous que le code de formulaire est placé dans les balises body de votre HTML
- Testez dans différents navigateurs pour éliminer les problèmes spécifiques au navigateur

**Les soumissions n'apparaissent pas dans ma liste**

- Confirmez que le formulaire est connecté à la bonne liste de contacts
- Vérifiez si le double opt-in est activé (les abonnés doivent confirmer)
- Vérifiez que votre liste n'a pas atteint sa limite d'abonnés
- Recherchez les soumissions dans le statut en attente ou non confirmé

**Le style du formulaire ne correspond pas à mon site web**

- Utilisez les options de personnalisation CSS dans l'éditeur de formulaire
- Ajoutez du CSS personnalisé pour remplacer les styles par défaut
- Vérifiez si le CSS de votre site web entre en conflit avec les styles de formulaire
- Considérez utiliser l'option d'intégration iframe pour une meilleure isolation

**Messages d'erreur lors de la soumission du formulaire**

- Assurez-vous que tous les champs requis sont remplis correctement
- Vérifiez si la validation de email bloque certains formats de email
- Vérifiez que le formulaire n'a pas été modifié après l'intégration
- Testez avec différentes adresses email pour identifier des patterns
