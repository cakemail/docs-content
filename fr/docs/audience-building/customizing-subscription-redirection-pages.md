# Dépannage des formulaires

## Description

Dans cet article, nous vous montrerons comment rediriger vos abonnés vers la page de destination de votre choix après qu'ils :
- S'inscrivent
- Confirment leur abonnement  
- Se désabonnent

## Pourquoi c'est important

Vos contacts peuvent recevoir une expérience plus personnalisée lorsqu'ils s'abonnent et se désabonnent de votre liste. Vous pouvez ajouter votre propre logo à la page de destination, la rendant plus élégante, plus claire et en accord avec votre marque.

## Instructions étape par étape

Si vous avez une liste existante, vous pouvez activer cette fonctionnalité dans les paramètres de votre liste. Voici comment procéder :

## Remarques importantes

- **Nouvelles listes** : Cette fonctionnalité est automatiquement déployée pour les nouvelles listes

- **Image de marque personnalisée** : Vous pouvez également utiliser vos propres pages d'abonnement, de confirmation d'abonnement et de désabonnement marquées simplement en modifiant l'URL

- **Exigence de logo** : Assurez-vous d'avoir téléchargé un logo dans votre profil d'entreprise pour un marquage optimal

## Types de pages de redirection

Vous pouvez personnaliser les pages de redirection pour :
- **Confirmation d'abonnement** : Après qu'une personne s'inscrive
- **Confirmation de double acceptation** : Après qu'une personne confirme son abonnement
- **Confirmation de désabonnement** : Après qu'une personne se désabonne

## Avantages

- **Cohérence de marque** : Maintenez l'expérience de votre marque tout au long du processus d'abonnement
- **Apparence professionnelle** : Les pages personnalisées paraissent plus professionnelles que les pages système par défaut
- **Meilleure expérience utilisateur** : Les abonnés voient une image de marque et des messages familiers
- **Établissement de confiance** : Une image de marque cohérente renforce la confiance avec votre audience

## Dépannage

**Le logo n'apparaît pas sur les pages de redirection**

- Assurez-vous d'avoir téléchargé un logo dans les paramètres de votre profil d'entreprise
- Vérifiez le format du fichier du logo (JPG, PNG recommandés)
- Vérifiez que la taille du fichier du logo n'est pas trop importante (moins de 500 Ko)
- Videz le cache de votre navigateur et rechargez la page

**L'URL personnalisée ne redirige pas correctement**

- Vérifiez que l'URL inclut le protocole https:// ou http://
- Vérifiez que la page de destination est en ligne et accessible
- Assurez-vous qu'il n'y a pas de fautes de frappe dans l'URL
- Testez l'URL directement dans un navigateur avant d'enregistrer

**Les changements ne se reflètent pas sur les pages de redirection**

- Accordez quelques minutes pour que les changements se propagent
- Videz le cache et les témoins de votre navigateur
- Essayez de visualiser dans une fenêtre de navigation incognito/privée
- Vérifiez que vous avez cliqué sur Enregistrer après avoir apporté des modifications

**Différentes pages s'affichent pour différentes actions**

- Vérifiez que vous avez défini l'URL correcte pour chaque type d'action
- Vérifiez les paramètres d'abonnement par rapport aux paramètres de désabonnement séparément
- Assurez-vous que les paramètres de double acceptation correspondent à votre flux prévu
- Testez chaque scénario (s'abonner, confirmer, se désabonner) individuellement
