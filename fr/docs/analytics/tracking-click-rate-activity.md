# Suivre l'activité du taux de clics

## Description

Vous pouvez suivre la performance de clics par URL pour obtenir des détails supplémentaires sur lesquels de vos contacts ont cliqué sur quels liens dans votre campagne. Vous pouvez aussi surveiller combien de fois ils ont cliqué sur le lien ainsi que quand et sur quel type d'appareil.

## Pourquoi c'est important

Les détails supplémentaires concernant votre taux de clics dans vos campagnes vous permettront d'analyser ces contacts individuellement. Vous pouvez ensuite utiliser ces données pour créer des segments, par exemple, pour envoyer des campagnes ciblées aux parties intéressées. Voici comment segmenter vos listes.

## Processus étape par étape

### Campagne envoyée

<Stepper>

1. Cliquez sur Campagnes

1. Cliquez sur la campagne pour laquelle vous voulez évaluer la performance.

1. Sélectionnez votre campagne envoyée.
![Sélectionner votre campagne envoyée](/images/analytics/9AB10ABA-6EB3-4435-B1DD-E436E1359F45_4_5005_c.jpeg)

1. Sous Performance, cliquez sur **Performance par URL**.

1. Cliquez sur **Afficher tout**.

1. Cliquez sur **# Personnes ont visité cette URL** pour savoir qui a cliqué sur le lien.
![Performance de clics par URL](/images/analytics/Screen_Shot_2020-10-26_at_12.53.08_PM.png)

1. Sous Activité des contacts, sélectionnez l'onglet **Cliqué** pour voir qui a cliqué sur les liens.
![Voir les personnes qui ont visité l'URL](/images/analytics/Screen_Shot_2020-10-26_at_12.56.43_PM.png)
   Optimisez votre stratégie de marketing par email en prenant cette information en compte et vous obtiendrez de meilleurs résultats.
</Stepper>


## Dépannage

**Aucune donnée de clic affichée pour les liens**

- Vérifiez que le suivi de liens est activé dans vos paramètres de campagne
- Vérifiez si les liens sont correctement formatés (doivent commencer par http:// ou https://)
- Assurez-vous que les liens ne redirigent pas à travers des raccourcisseurs tiers qui bloquent le suivi
- Testez les liens en mode aperçu avant d'envoyer la campagne

**Les comptes de clics ne correspondent pas aux analyses de page de destination**

- Le suivi de clics de email et les analyses de site web peuvent utiliser des méthodes de comptage différentes
- Vérifiez les clics de robots des scanners de sécurité (taux de clics inhabituellement élevés immédiatement après l'envoi)
- Examinez si plusieurs redirections causent des divergences de suivi
- Considérez les différences de fuseau horaire entre les plateformes

**Impossible d'exporter les données d'activité de clics**

- Vérifiez que vous avez les permissions de compte nécessaires pour l'exportation de données
- Vérifiez s'il y a une limite sur le nombre d'enregistrements que vous pouvez exporter
- Essayez d'exporter des plages de dates plus petites si vous traitez de grands ensembles de données
- Assurez-vous que votre navigateur permet les téléchargements de fichiers depuis la plateforme
