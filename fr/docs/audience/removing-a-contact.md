# Supprimer un contact

## Description

Les contacts peuvent choisir de se désinscrire de votre liste, ils peuvent le faire en cliquant sur un lien [SE DÉSINSCRIRE] dans une campagne de courriel que vous leur avez envoyée, ou cela peut être fait manuellement (un contact peut vous envoyer un courriel avec une demande de désinscription, par exemple). Apprenez comment ajouter un lien de désinscription à votre courriel. De même, vous pouvez supprimer définitivement les informations d'un utilisateur de votre liste. Nous expliquerons comment le faire dans cet article des deux façons.

**Deux options disponibles :**
- Désinscrire un contact
- Supprimer définitivement un contact

## Pourquoi c'est important

Si vous traitez manuellement les demandes de désabonnement correctement, vous pouvez maintenir une bonne réputation d'expéditeur. Par conséquent, votre délivrabilité ne sera pas affectée. **Attention :** Différenciez entre Supprimer vs désabonner un contact.

## Instructions étape par étape

### Désabonner un contact

Pour désabonner un contact manuellement, suivez ces étapes :

<Stepper>

1. Sélectionnez l'onglet **Contacts**
![Sélectionner l'onglet Contacts](/images/audience/082ED574-5063-48A2-88B4-7AC675D1674A_4_5005_c.jpeg)

1. Choisissez votre liste
![Choisir votre liste](/images/audience/CE3A5EE2-085F-4616-B867-C695822E58D2_4_5005_c.jpeg)

1. Trouvez, avec la loupe, le email du contact
![Rechercher un contact](/images/audience/7FD16F10-C04E-46BA-8FB9-E1384E2C1AA2_4_5005_c.jpeg)

1. Cliquez sur les **...** dans la ligne du contact
![Cliquer sur le menu à trois points](/images/audience/A7006973-F052-4FBB-A9CD-195908E887AC_4_5005_c.jpeg)

1. Sélectionnez **Désabonner**
![Sélectionner l'option Désabonner](/images/audience/35383A02-3052-455E-A24E-3F840643410F_4_5005_c.jpeg)
*Vous pourrez également éditer, supprimer ou ajouter ce contact à la liste de suppression.*

1. Le message de désabonnement apparaîtra en haut de votre page pour confirmer l'action
</Stepper>

### Supprimer définitivement un contact

Vous pouvez également supprimer des contacts de votre liste définitivement. Notez que toutes les informations seront perdues. Voici comment procéder :

<Stepper>

1. Sélectionnez l'onglet **Contacts**
![Sélectionner l'onglet Contacts](/images/audience/082ED574-5063-48A2-88B4-7AC675D1674A_4_5005_c.jpeg)

1. Choisissez votre liste
![Choisir votre liste](/images/audience/CE3A5EE2-085F-4616-B867-C695822E58D2_4_5005_c.jpeg)

1. Trouvez le contact avec la recherche à la loupe
![Rechercher un contact](/images/audience/7FD16F10-C04E-46BA-8FB9-E1384E2C1AA2_4_5005_c.jpeg)

1. Cliquez sur les **...** dans la ligne du contact
![Cliquer sur le menu à trois points](/images/audience/A7006973-F052-4FBB-A9CD-195908E887AC_4_5005_c.jpeg)

1. Sélectionnez **Supprimer**
![Sélectionner l'option Supprimer](/images/audience/35383A02-3052-455E-A24E-3F840643410F_4_5005_c.jpeg)

1. Confirmez la suppression définitive
![Dialogue de confirmation de suppression](/images/audience/Screen_Shot_2020-11-09_at_3.00.59_PM.png)
</Stepper>

## Considérations importantes

- **Désabonner vs Supprimer** : Le désabonnement maintient les dossiers pour la conformité, tandis que la suppression supprime définitivement toutes les données
- **Conformité** : Honorez toujours les demandes de désabonnement rapidement pour maintenir une bonne réputation d'expéditeur
- **Perte de données** : La suppression définitive ne peut pas être annulée - utilisez avec précaution

## Dépannage

**Impossible de désabonner le contact**

- Vérifiez si le contact est déjà désabonné
- Vérifiez que vous avez les bonnes permissions
- Assurez-vous que le contact existe dans la liste sélectionnée
- Essayez d'utiliser l'option de désabonnement manuel

**Le contact désabonné reçoit encore des emails**

- Vérifiez si le contact est dans plusieurs listes
- Vérifiez que le désabonnement a été appliqué à toutes les listes
- Accordez 24-48 heures pour la propagation complète
- Vérifiez les emails transactionnels actifs

**Impossible de supprimer le contact définitivement**

- Assurez-vous que le contact ne fait pas partie d'une automatisation active
- Vérifiez si la suppression est bloquée par les règles de conformité
- Vérifiez que vous avez les permissions d'administrateur
- Essayez de désabonner d'abord, puis de supprimer

**Le contact supprimé réapparaît**

- Vérifiez si le contact est en cours de ré-importation
- Vérifiez que la suppression a été confirmée correctement
- Cherchez des entrées en double avec des variations
- Révisez les journaux d'importation récents

**Le lien de désabonnement ne fonctionne pas pour le contact**

- Vérifiez que le lien inclut les bonnes balises de fusion
- Vérifiez que la campagne a été envoyée depuis la bonne liste
- Assurez-vous que la page de désabonnement fonctionne
- Testez avec un client email différent

**Le contact demande à se réabonner**

- Utilisez l'option d'édition de contact pour changer le statut
- Assurez-vous d'avoir le consentement écrit
- Documentez la demande de réabonnement
- Vérifiez si le contact est dans la liste de suppression
