# Définir votre fuseau horaire

## Description

Pour vous assurer que vos courriels sont programmés selon votre heure préférée, vous devez définir votre fuseau horaire avec précision. Ce guide vous guide à travers les étapes pour changer votre fuseau horaire.

## Pourquoi c'est important

Le timing est essentiel dans le marketing par courriel. Assurez-vous que vos courriels sont envoyés quand vous voulez qu'ils le soient.

## Instructions étape par étape

Accéder aux paramètres de votre compte

<Stepper>

1. Cliquez sur le nom de votre entreprise/adresse courriel dans le coin supérieur droit (ceci ouvre un menu déroulant), puis cliquez sur Utilisateurs

1. Cliquez sur le compte utilisateur dont vous voulez changer le fuseau horaire.

1. Ajustez le fuseau horaire
Sur la page utilisateur, localisez l'option fuseau horaire. Cliquez sur le menu déroulant et sélectionnez votre fuseau horaire préféré parmi les options disponibles.

1. Sauvegardez les changements
Après avoir sélectionné le bon fuseau horaire, cliquez sur **Sauvegarder** pour appliquer les changements.
![Sauvegarder les changements de fuseau horaire](/images/accounts/accounts-setting-up-your-timezone-133.png)
Notez que changer votre fuseau horaire n'affecte que l'heure de programmation de votre compte, pas le fuseau horaire du destinataire. Considérez le fuseau horaire de votre public cible lors de la programmation de campagnes de courriel.
Définir votre fuseau horaire avec précision assure que vos campagnes de courriel sont programmées à votre heure préférée. En suivant ces étapes, vous pouvez ajuster votre fuseau horaire pour correspondre à votre localisation et programmer des campagnes efficacement. N'oubliez pas de considérer le fuseau horaire de votre public cible pour un timing de livraison optimal.
</Stepper>

## Dépannage

**Les changements de fuseau horaire ne prennent pas effet**
- Sauvegardez les changements en cliquant sur le bouton "Sauvegarder"
- Déconnectez-vous et reconnectez-vous
- Videz le cache du navigateur
- Vérifiez les campagnes programmées pour les mises à jour

**Impossible de trouver mon fuseau horaire**
- Recherchez votre ville principale la plus proche
- Utilisez le décalage UTC si la ville n'est pas listée
- Vérifiez les options d'heure d'été
- Contactez le support pour les fuseaux horaires manquants

**Les courriels programmés s'envoient à la mauvaise heure**
- Vérifiez que votre paramètre de fuseau horaire est correct
- Vérifiez les ajustements de l'heure d'été
- Révisez les paramètres de programmation de campagne
- Considérez les fuseaux horaires des destinataires

**Plusieurs utilisateurs avec des fuseaux horaires différents**
- Chaque utilisateur doit définir son propre fuseau horaire
- Le fuseau horaire au niveau du compte affecte les rapports
- La programmation de campagne utilise le fuseau horaire de l'expéditeur
- Coordonnez les horaires d'équipe en conséquence

**Le fuseau horaire revient au paramètre précédent**
- Assurez-vous d'avoir la permission de modifier les paramètres
- Vérifiez l'interférence du remplissage automatique du navigateur
- Essayez d'utiliser un autre navigateur
- Contactez le support si le problème persiste
