# Mise à jour de mon profil d'entreprise

## Description

Les paramètres de votre profil d'entreprise doivent être à jour avant d'envoyer votre première campagne, car la loi exige une adresse physique dans chaque campagne.

## Pourquoi c'est important

- **Conformité légale** : L'adresse physique est requise dans chaque campagne par la loi
- **Cohérence de marque** : Le logo de l'entreprise peut être utilisé dans vos campagnes de email
- **Apparence professionnelle** : Les informations complètes du profil inspirent confiance
- **Informations de contact** : Fournit aux destinataires des détails d'entreprise légitimes

## Instructions étape par étape

Pour mettre à jour le profil de votre entreprise, suivez les instructions.

<Stepper>

1. Cliquez sur le nom de votre entreprise/adresse email dans le coin supérieur droit (cela ouvre un menu déroulant), puis cliquez sur Compte

1. Cliquez sur le **bouton crayon** pour modifier chaque partie
</Stepper>

## Champs que vous pouvez ajouter/modifier :

Voici une liste des champs que vous pouvez ajouter/modifier :

- **Logo de l'entreprise** (peut être utilisé dans vos emails)
- **Nom légal de l'entreprise**
- **Adresse** 
- **Numéro de téléphone**
- **Site web**

## Meilleures pratiques

- **Gardez les informations à jour** : Mettez à jour les détails chaque fois que les informations de votre entreprise changent
- **Complétez tous les champs** : Remplissez tous les champs pertinents pour une apparence professionnelle
- **Logo de haute qualité** : Utilisez une image de logo claire et professionnelle
- **Adresse exacte** : Assurez-vous que votre adresse physique est correcte et actuelle
- **Informations de contact valides** : Fournissez des numéros de téléphone qui fonctionnent et des URL de site web

## Dépannage

**Impossible de télécharger le logo de l'entreprise**

- Vérifiez la taille du fichier (max 5 Mo)
- Utilisez le format PNG ou JPG
- Assurez-vous que les dimensions de l'image sont raisonnables
- Essayez de réduire la résolution de l'image

**Erreurs de validation d'adresse**
- Incluez tous les champs requis
- Utilisez le format d'adresse standard
- Évitez les caractères spéciaux
- Vérifiez le format du code postal

**Les changements ne se sauvegardent pas**
- Cliquez sur "Sauvegarder" après chaque modification
- Vérifiez les erreurs de validation
- Assurez-vous que tous les champs requis sont remplis
- Essayez de modifier un champ à la fois

**Le logo n'apparaît pas dans les emails**
- Publiez les changements après le téléchargement
- Vérifiez que le modèle de email inclut la balise de fusion du logo
- Vérifiez que l'URL du logo est accessible
- Videz le cache du client de email

**Impossible de modifier les informations de l'entreprise**
- Confirmez que vous avez les privilèges d'administrateur
- Vérifiez si le compte est suspendu
- Assurez-vous que les champs ne sont pas verrouillés par l'organisation
- Contactez le propriétaire du compte pour les permissions

**URL du site web non acceptée**
- Incluez l'URL complète (https://...)
- Supprimez les espaces de fin
- Assurez-vous que l'URL est publiquement accessible
- Utilisez le format de domaine standard
