# Télécharger vers le gestionnaire de fichiers

## Description

Votre bibliothèque vous permet de télécharger des fichiers tels que des images ou des PDF. Par exemple, vous pouvez stocker les images qui vont avec votre marque et structure établies. Vous pouvez utiliser ceci de plusieurs façons à travers vos campagnes de email.

## Pourquoi c'est important
Avoir toutes vos images et fichiers au même endroit vous aidera à vous préparer à envoyer vos campagnes et les améliorer avec d'excellents visuels ou du contenu utile.

## Instructions étape par étape

La procédure pour ajouter des fichiers au gestionnaire de fichiers est la suivante :

### Pour les images dans le contenu de email :
<Stepper>

1. Sélectionnez **Modifier le email** dans la barre de campagne
![Emplacement du bouton de modification de campagne](/images/campaign-creation/A5E92D9F-D74F-48BA-A833-017C5A80467E_4_5005_c.jpeg)
![Bouton Modifier le email dans la barre de campagne](/images/campaign-creation/558DDDA4-0A8D-4A95-8086-0D375ABAA3FB_4_5005_c.jpeg)

1. Dans la section Aperçu du email, sélectionnez **Modifier le email**
![Bouton Modifier le email dans la section aperçu](/images/campaign-creation/C70DF271-09A8-4909-834F-2D307160106F_4_5005_c.jpeg)

1. Dans le menu Contenu, sélectionnez **IMAGE** et glissez le bouton vers le contenu du email
![Glisser le bloc de contenu image vers le email](/images/campaign-creation/8F54E671-7BCF-42E9-A4B2-608DB413E3A8.png)

1. Cliquez sur **Parcourir**
![Bouton Parcourir pour la sélection d'image](/images/campaign-creation/04BF58E7-BBE8-450A-85CF-FB37F960B0D0_4_5005_c.jpeg)

1. Vous avez maintenant trois options pour ajouter des fichiers :
   - **Télécharger** - Téléchargez vos propres fichiers
   - **Importer** - Importez depuis des sources externes
   - **Parcourir** - Parcourez depuis la sélection de photos gratuites.
![Interface d'options de téléchargement de fichier](/images/campaign-creation/D18A765E-5CA0-4C5A-9117-48B46310CB81_4_5005_c.jpeg)

1. Cliquez sur **Télécharger** puis sélectionnez votre fichier

1. Votre fichier est maintenant dans notre gestionnaire de fichiers et peut être sélectionné à partir de là
**Note :** Actuellement, vous ne pouvez télécharger qu'un fichier à la fois (aucun FTP n'est disponible)
</Stepper>

### Pour les liens de fichier :
<Stepper>

1. Cliquez sur le **bouton de lien** dans la section où vous voulez inclure votre fichier.

1. Utilisez le menu déroulant de type d'URL, choisissez **Gestionnaire de fichiers**

1. Sélectionnez le fichier que vous avez téléchargé plus tôt depuis votre bibliothèque.

1. Le fichier est maintenant lié à l'intérieur du email.

1. Les destinataires peuvent y accéder en cliquant sur le lien
</Stepper>

## Limitations de fichier
- **Taille d'image maximale** : 1600px par 2000px
- **Taille de fichier maximale** : 1 Mo

## Dépannage

**Erreur d'échec de téléchargement**

- Vérifiez que la taille du fichier est sous la limite de 1 Mo
- Vérifiez que les dimensions d'image ne dépassent pas 1600x2000px
- Assurez-vous que le format de fichier est supporté (JPG, PNG, GIF, PDF)

**Le fichier n'apparaît pas dans la bibliothèque**

- Actualisez la page du navigateur après le téléchargement
- Videz le cache du navigateur si les fichiers manquent
- Vérifiez l'espace de stockage disponible dans votre compte

**Problèmes de qualité d'image**

- Optimisez les images avant de les télécharger (72-150 DPI pour le web)
- Utilisez JPG pour les photos, PNG pour les graphiques avec transparence
- Compressez les gros fichiers sans perdre la qualité visible

**Impossible de sélectionner le fichier téléchargé**

- Attendez que le téléchargement se termine complètement
- Vérifiez les permissions et propriété de fichier
- Essayez de re-télécharger si le fichier semble corrompu

**Limite de stockage atteinte**

- Supprimez les fichiers inutilisés de la bibliothèque
- Compressez les fichiers avant de les télécharger
- Contactez le support pour augmenter la capacité de stockage

**Vitesse de téléchargement lente**

- Vérifiez la stabilité de la connexion Internet
- Téléchargez pendant les heures creuses
- Brisez les gros téléchargements en plus petits lots
