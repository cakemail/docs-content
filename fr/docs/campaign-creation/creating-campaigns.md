# Créer des campagnes

<iframe src="//www.youtube-nocookie.com/embed/U9Rsh0TNrJE" width="560" height="315" frameborder="0" allowfullscreen=""></iframe>

## Description

Les campagnes de marketing par courriel consistent en un courriel envoyé simultanément à plusieurs personnes avec un objectif spécifique ; les entreprises du monde entier les utilisent pour les aider à vendre, à se connecter et à atteindre leurs objectifs. À long terme, vous pouvez vous efforcer d'amener vos abonnés à s'inscrire à un webinaire ou à acheter quelque chose sur votre site, mais à court terme, vous devez principalement vous concentrer sur l'intérêt du lecteur et renforcer votre relation. Lisez-en plus à ce sujet dans notre article sur la stratégie de contenu.

## Pourquoi c'est important

La majorité des gens consultent leur courriel plusieurs fois par jour, il est donc logique d'envoyer des courriels à vos clients au sujet de votre produit. Une relation directe avec votre audience ne peut être établie qu'à travers cela. Par conséquent, le marketing par courriel est la meilleure façon de cultiver les prospects. En quelques clics, vous pouvez vous connecter avec des centaines, des milliers, voire plus de clients.

Du texte, des images, des vidéos, des animations et bien plus encore peuvent tous être utilisés pour transmettre votre message. Vos résultats seront suivis, comme le nombre de personnes qui ont ouvert votre campagne, cliqué dessus, combien ont été marquées comme spam, etc. En fin de compte, c'est une stratégie marketing rentable éprouvée.

## Instructions étape par étape
Vous pouvez envoyer une campagne à vos destinataires avec notre application de marketing par email. Vous avez plusieurs options disponibles lors de la création d'une campagne ; voici un bref aperçu de la façon de commencer.

Pour commencer le processus de création, sélectionnez l'onglet **Créer** dans le coin supérieur droit de l'application, et cliquez sur **Campagne**.

![Capture d'écran de l'interface de création](/images/campaign-creation/4404612263707.jpeg)
Vous pouvez ensuite choisir parmi les options suivantes :

- Utiliser vos propres modèles - réguliers ou déjà marqués
- Utiliser un modèle professionnel
- Réutiliser une campagne précédemment envoyée
- Concevoir votre propre modèle

Une fois que vous avez sélectionné une option, vous serez dirigé vers le concepteur de email où vous pourrez personnaliser et modifier votre modèle. Apprenez-en plus sur le concepteur de email.

Si vous avez **marqué** votre compte, en téléchargeant votre logo et en incluant votre code couleur, vous aurez accès aux **Modèles intelligents**, une sélection de modèles créés pour afficher automatiquement votre marque afin de rendre votre email soigné et professionnel.

### Utiliser vos propres modèles

Vous avez peut-être construit ou marqué un modèle et souhaitez l'utiliser à nouveau. Voici comment vous pouvez le faire :

<Stepper>

1. Cliquez sur l'onglet **Mes modèles**.
![Interface de filtre et de recherche des modèles professionnels](/images/campaign-creation/4404618523419.jpeg)

1. Sélectionnez l'option **Aperçu** sous votre modèle préféré.
![Bouton d'option d'aperçu du modèle](/images/campaign-creation/4404618602011.jpeg)

1. Cliquez sur **Sélectionner le modèle** dans le coin supérieur droit de la fenêtre d'aperçu du email.
![Bouton de sélection du modèle dans la fenêtre d'aperçu](/images/campaign-creation/4404613244955.jpeg)

1. Vous pouvez filtrer par type d'éditeur : Éditeur HTML ou Concepteur de email.
![](/images/campaign-creation/Dec-21-2021%2016-06-46.gif)

1. Modifiez votre modèle sauvegardé
Le modèle que vous modifiez peut être sauvegardé plus tard avec vos modifications pour être utilisé comme futur modèle spécialement marqué et conçu pour répondre à vos besoins spécifiques.
</Stepper>

![Options de modèles à partir de zéro](/images/campaign-creation/4404613821595.jpeg)

</Stepper>

### Concevoir votre propre modèle

Si vous vous sentez à l'aise avec votre talent de conception ou avez un spécialiste qui peut le faire, vous pouvez facilement créer votre propre modèle. Mais attention : concevoir un email qui s'affiche correctement dans tous les navigateurs est délicat ! Pour créer votre propre modèle à partir de zéro, sélectionnez l'onglet **Commencer à partir de zéro**.

![](/images/campaign-creation/A6FE63E4-E729-4627-88AF-18766D4C3357_4_5005_c.jpeg)

Vous pouvez ensuite choisir parmi ces trois options différentes :

- Commencer avec l'essentiel
- Commencer avec une toile vierge
- Commencer avec votre propre code.

#### Commencer avec l'essentiel
![](/images/campaign-creation/Start%20with%20the%20essentials.png)

Choisissez cette option pour ouvrir l'éditeur avec un modèle de base que vous pouvez ensuite modifier selon vos souhaits. Apprenez-en plus sur l'éditeur ici.

#### Commencer avec une toile vierge

Sélectionnez un modèle vierge et utilisez votre imagination pour créer votre contenu.
![](/images/campaign-creation/Start%20with%20a%20blank%20canvas.png)
#### Commencer avec votre propre code

Si vous voulez importer du code HTML écrit en dehors de notre outil, vous pouvez le coller dans notre éditeur avec cette option. Le email peut être modifié en utilisant un éditeur HTML.

Voici comment vous pouvez ouvrir l'éditeur :

1. Sélectionnez l'option **Commencer avec votre propre code**
2. Collez votre code sur la partie noire de la page
3. Cliquez sur **Télécharger et modifier**


![](/images/campaign-creation/Paste%20your%20code.png)

Avant d'envoyer votre campagne, il est essentiel de savoir exactement comment elle apparaîtra sur différentes plateformes. Avec la fonctionnalité d'aperçu de campagne, vous pouvez faire exactement cela ! Que ce soit sur le web, mobile ou en mode sombre, vous pouvez voir votre campagne exactement comme vos clients la verront. ## Tester et envoyer votre campagne
Quel que soit le choix que vous avez fait pour vos modèles, il est très important de tester votre campagne minutieusement avant d'envoyer des emails à vos clients, surtout avec un nouveau design. Testez-la sur différents navigateurs de email et n'oubliez pas de vérifier sur votre téléphone aussi. Afficher un modèle sur n'importe quelle plateforme peut être complexe ! Mettez-y une ligne d'objet, choisissez une heure et une date, assoyez-vous et regardez les résultats !

Vous pouvez utiliser divers sujets comme contenu pour votre campagne. Nous vous suggérons de nous contacter si vous n'êtes toujours pas sûr de ce qu'il faut utiliser pour votre contenu, nous serons heureux de discuter de votre entreprise et de vous aider à commencer. Vous pouvez également lire nos différents articles à ce sujet sur notre blog.

## Dépannage

**Le concepteur de email ne se charge pas**

- Videz le cache et les cookies de votre navigateur
- Essayez d'utiliser Chrome ou Firefox pour une meilleure compatibilité
- Désactivez les extensions de navigateur qui pourraient interférer

**Le modèle ne se sauvegarde pas**

- Vérifiez la stabilité de votre connexion Internet
- Sauvegardez fréquemment lors du travail sur des designs complexes
- Utilisez la fonction de sauvegarde automatique si disponible

**Les images ne s'affichent pas**

- Assurez-vous que les images font moins de 1 Mo pour un chargement optimal
- Utilisez des formats supportés (JPG, PNG, GIF)
- Téléchargez les images dans la bibliothèque avant de les insérer

**La campagne reste en brouillon**

- Vérifiez que tous les champs requis sont complétés
- Vérifiez qu'une liste de destinataires est sélectionnée
- Assurez-vous que la ligne d'objet et les informations de l'expéditeur sont remplies

**Problèmes d'importation HTML**

- Validez votre code HTML avant l'importation
- Supprimez les références JavaScript et CSS externes
- Utilisez du CSS en ligne pour une meilleure compatibilité avec les clients de email

**L'aperçu ne correspond pas au email final**

- Testez l'envoi à plusieurs adresses email
- Vérifiez le rendu dans les clients de email réels
- Tenez compte des limitations des clients de email dans la conception
