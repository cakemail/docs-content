# Supprimer un modèle

## Description

La gestion des modèles est une partie essentielle du maintien d'un flux de travail de marketing par email efficace. Cette fonctionnalité vous permet de supprimer définitivement les modèles de email indésirables ou obsolètes de votre compte Cakemail. Que vous nettoyiez des modèles de test, supprimiez des designs saisonniers, ou simplement désencombreriez votre espace de travail, le processus de suppression de modèles garantit que votre bibliothèque de modèles reste organisée et ne contient que des designs pertinents et actuels qui s'alignent avec votre marque et vos besoins marketing.

## Pourquoi c'est important

Rester organisé est toujours important ; les choses seront plus faciles à trouver et ainsi de suite si l'environnement de travail est bien maintenu. Si votre zone de travail est encombrée de copies des modèles que vous avez créés et que vous aimeriez la nettoyer, suivez les instructions ci-dessous.

## Instructions étape par étape

Suivez ces instructions pour supprimer un modèle :

<Stepper>

1. **Sélectionnez Campagnes** dans le menu du haut
![Sélectionner Campagnes dans le menu du haut](/images/email-templates/A5E92D9F-D74F-48BA-A833-017C5A80467E_4_5005_c.jpeg)

1. **Sélectionnez l'onglet Modèles** dans la page Campagnes
![Sélectionner l'onglet Modèles](/images/email-templates/A8326E57-D507-41E9-9294-7514E67F697C_4_5005_c.jpeg)

1. **Cliquez sur la flèche** à côté de Créer une campagne
![Cliquer sur la flèche à côté de Créer une campagne](/images/email-templates/BFC70ED6-FE59-4E7B-8BFC-34F56CC6A85C_4_5005_c.jpeg)

1. **Sélectionnez Supprimer**
![Sélectionner l'option Supprimer](/images/email-templates/73274CD3-5757-41E9-AC3B-C91401915DD5_4_5005_c.jpeg)

1. **Entrez le mot DELETE** dans le champ vide (en majuscules)

1. **Cliquez sur Supprimer**
![Entrer DELETE et cliquer sur le bouton Supprimer](/images/email-templates/Screen_Shot_2021-07-06_at_11.21.16_AM.png)
Maintenant que votre modèle a été supprimé, vous avez un espace de travail propre.
</Stepper>

## Notes importantes
- **Action permanente** : La suppression de modèle ne peut pas être annulée
- **Confirmation requise** : Vous devez taper "DELETE" en majuscules pour confirmer
- **Organisation** : Un nettoyage régulier aide à maintenir un espace de travail organisé
- **Aucun impact sur les campagnes envoyées** : Supprimer un modèle n'affectera pas les campagnes déjà envoyées utilisant ce modèle

## Quand supprimer des modèles
- **Designs obsolètes** : Supprimez les modèles qui ne correspondent plus à votre marque
- **Modèles dupliqués** : Nettoyez les copies inutiles
- **Modèles saisonniers** : Supprimez les modèles qui ne sont plus pertinents
- **Modèles de test** : Supprimez les modèles expérimentaux ou de test
- **Gestion de l'espace de travail** : Gardez seulement les modèles activement utilisés

## Meilleures pratiques

- **Examinez avant de supprimer** : Assurez-vous que vous n'aurez pas besoin du modèle à l'avenir
- **Exportez si nécessaire** : Sauvegardez une copie du code HTML si vous pourriez vouloir le recréer
- **Nettoyage régulier** : Examinez et nettoyez périodiquement votre bibliothèque de modèles
- **Convention de nommage** : Utilisez des noms clairs pour identifier facilement les modèles à supprimer

## Dépannage

**J'ai tapé DELETE mais rien ne se passe**

- Assurez-vous d'avoir tapé "DELETE" en lettres majuscules
- Vérifiez les espaces supplémentaires avant ou après le mot
- Vérifiez que vous avez cliqué sur le bouton de confirmation après avoir tapé
- Essayez d'utiliser un navigateur différent si le problème persiste

**J'ai accidentellement supprimé le mauvais modèle**

- Malheureusement, la suppression est permanente et ne peut pas être annulée
- Vérifiez si vous avez une sauvegarde ou une version exportée du HTML
- Recherchez des modèles similaires que vous pouvez dupliquer et modifier

**Je ne trouve pas les modèles à supprimer**

- Assurez-vous d'être dans le bon compte ou sous-compte
- Vérifiez toutes les catégories et filtres de modèles
- Vérifiez que les modèles ne sont pas cachés par les filtres de recherche
- Regardez dans les sections "Mes modèles" et modèles partagés
