# Gestion de votre facturation

**Source:** [Assistance Cakemail](https://support.cakemail.com/hc/en-us/articles/1500000251681-Managing-your-billing)  
**Auteur :** Selma Aboudou
**Dernière mise à jour :** il y a 9 mois

## Description

![Capture d'écran de l'interface](/images/billing/billing-managing-your-billing-395.png)


Nos options de compte peuvent être adaptées selon vos préférences et besoins. Ceci vous guidera à travers le processus simple de gestion de la facturation de votre compte.

## Pourquoi c'est important

![Capture d'écran de l'interface](/images/billing/billing-managing-your-billing-396.png)


Nous proposons différentes options de compte pour répondre à vos besoins. Avec le compte gratuit, vous pourrez interagir avec votre audience immédiatement, et avec les forfaits de croissance, vous pourrez continuer à développer votre entreprise. Si votre compte dépasse le plan gratuit, vous devez sélectionner l'un de nos forfaits de croissance. De plus, nos forfaits premium sont disponibles pour les entreprises qui nécessitent des outils plus spécialisés.

## Pour accéder à la facturation de votre compte

![Capture d'écran de l'interface](/images/billing/billing-managing-your-billing-397.png)


Dans le coin supérieur droit de l'écran, cliquez sur la flèche à côté de votre nom d'utilisateur


2. Cliquez sur le bouton **Changer de forfait** qui apparaît


3. Sélectionnez le forfait que vous souhaitez prendre ou votre forfait actuel pour ouvrir le portail de facturation


4. Cliquez sur le bouton **Mettre à niveau le forfait** sur la page Facturation pour voir nos différents forfaits dans le compte


5. Veuillez sélectionner votre forfait souhaité en cliquant sur **Continuer** à côté


6. En cliquant sur **Confirmer**, vous acceptez votre plan


7. Après cela, vous devez remplir les informations. Cela activera votre compte avec les limites appropriées afin que vous soyez facturé mensuellement à cette date



Il est possible d'effectuer une mise à niveau jusqu'à 75000 contacts par vous-même. Si vous devez ajouter plus de 75000 contacts, veuillez nous contacter afin que nous puissions vous aider. En cas de rétrogradation, votre forfait peut être modifié si votre liste a diminué. La facture du mois prochain sera calculée au prorata.

## Annulation d'abonnement

![Capture d'écran de l'interface de clic](/images/billing/billing-managing-your-billing-398.png)


Dans l'onglet **Facturation**, vous pouvez fermer votre compte en cliquant sur **Annuler l'abonnement**. Ensuite, vous pouvez commencer le processus d'annulation du compte. Une demande d'assistance automatique sera ouverte en votre nom pour garantir que le processus se déroule sans problème. Notre équipe d'assistance vous contactera pour confirmer les modifications et vous aider avec les étapes suivantes.
