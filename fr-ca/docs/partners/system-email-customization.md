# Personnalisation des courriels système

## Description

Les courriels système sont une partie clé de la gestion des comptes utilisateur, assurant une communication efficace entre votre plateforme et vos utilisateurs finaux. Ces courriels facilitent les processus de confirmation, les réinitialisations de mot de passe et autres actions importantes liées au compte.

## Pourquoi c'est important

Pour donner à vos clients une expérience transparente, il est essentiel de configurer ces communications avec précision et efficacité.

## Instructions étape par étape

### Accéder aux courriels système.

<Stepper>

1. Naviguez vers la section Compte dans votre tableau de bord.
![Naviguer vers la section Compte](/images/partners/partners-setting-up-and-customizing-system-emails-for-your-platform-169.png)

1. Cliquez sur Courriels système dans les options de menu.
Ici, vous gérerez tous les types de courriels automatisés que votre plateforme envoie.
![Option de menu Courriels système](/images/partners/e705db46-0036-4f02-8efd-2b7b64a091f0.png)
</Stepper>

### Créer un nouveau courriel système.

<Stepper>

1. Initier la création
Cliquez sur le bouton "+" dans le coin supérieur droit pour commencer à configurer un nouveau courriel système. Cela ouvrira une fenêtre pop-up avec divers champs à remplir.

1. Configurer les détails du courriel
Remplissez soigneusement chacun des champs suivants :
![Formulaire de configuration des détails du courriel](/images/partners/b36c7eb0-f7a9-4678-81d5-7ebf4a4c8954.png)

1. Examiner et confirmer
Examinez soigneusement toutes les informations que vous avez entrées pour vous assurer qu'elles sont correctes. Une fois satisfait, cliquez sur le bouton "Confirmer". Cela finalisera vos paramètres et activera le courriel système.
</Stepper>

## Comment fonctionnent les courriels système

Une fois configurés, les courriels système sont automatiquement envoyés aux utilisateurs finaux chaque fois qu'une action correspondante se produit, comme la vérification de compte ou les demandes de réinitialisation de mot de passe. Cette automatisation assure que les utilisateurs reçoivent des notifications importantes sans délai, rationalisant vos processus de communication.

Note : Pour que votre courriel système fonctionne, vous devez ajouter les balises de fusion appropriées pour déclencher l'action désirée. Vous pouvez le faire en suivant ces étapes :

1. Allez aux "Gabarits généraux"
2. Créez votre gabarit système puis modifiez-le
3. Ajoutez un appel à l'action dans votre courriel
4. Sélectionnez la zone de texte dans votre CTA et cliquez sur "Balises de fusion"

![Sélectionner le bouton balises de fusion dans CTA](/images/partners/d09abc41-5553-4f68-8015-d6fb089a4229.png)

5. Finalement, sélectionnez la balise de fusion appropriée à ajouter

![Sélectionner la balise de fusion appropriée de la liste](/images/partners/d01d153c-cbcd-47c3-8f0b-d34eb8df4233.png)

## Gérer et mettre à jour les courriels système

Vous pouvez toujours retourner à la section Courriels système pour examiner ou modifier vos configurations. Mettre à jour les courriels système périodiquement aide à garder votre contenu pertinent et aligné avec toute mise à jour ou changement à votre plateforme.

## Dépannage

**Les courriels système ne sont pas envoyés**

- Vérifiez que le courriel de l'expéditeur provient d'un domaine authentifié
- Vérifiez si les bonnes balises de fusion sont incluses dans le gabarit
- Assurez-vous que le type de courriel système est correctement configuré
- Examinez s'il y a des restrictions d'envoi au niveau du compte

**Les balises de fusion ne fonctionnent pas dans les gabarits**

- Confirmez que vous utilisez la syntaxe exacte des balises de fusion
- Vérifiez si la balise de fusion correspond au type de courriel système
- Vérifiez que les balises de fusion sont placées dans des éléments cliquables pour les actions
- Testez le gabarit avec des données d'aperçu avant d'activer

**Les utilisateurs reçoivent des courriels dans la mauvaise langue**

- Examinez le paramètre de langue pour chaque configuration de courriel système
- Vérifiez si les préférences de langue des utilisateurs remplacent les défauts
- Assurez-vous que les gabarits existent pour toutes les langues configurées
- Vérifiez que la langue du navigateur n'affecte pas la sélection

**Les modifications de gabarit ne se reflètent pas dans les courriels envoyés**

- Confirmez que les modifications ont été sauvegardées dans la section Gabarits généraux
- Vérifiez si vous devez mettre à jour la configuration du courriel système
- Accordez quelques minutes pour que les mises à jour de gabarit se propagent
- Videz le cache et vérifiez que vous modifiez le bon gabarit
