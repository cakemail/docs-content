# Authentification multi-facteurs (AMF)

## Description

L'authentification multi-facteurs (AMF) est une mesure de sécurité qui renforce la protection de votre compte. En activant l'AMF, vous améliorez activement la défense de votre compte contre l'accès non autorisé. Ce guide fournit des instructions étape par étape pour activer l'AMF sur votre compte.

## Pourquoi c'est important

Dans le monde numérique d'aujourd'hui, protéger vos comptes en ligne est essentiel. L'authentification multi-facteurs (AMF) fournit une couche supplémentaire de sécurité, réduisant le risque d'accès non autorisé et de violations potentielles de données. Mettre en place l'AMF est une étape proactive vers le renforcement de la sécurité de votre compte.

## Instructions étape par étape

<Stepper>

1. Cliquez sur le nom de votre entreprise/adresse courriel dans le coin supérieur droit (cela ouvre un menu déroulant), puis cliquez sur Utilisateurs

1. Choisissez l'utilisateur pour l'activation de l'AMF

1. Faites défiler vers le bas jusqu'à « Sécurité » et cliquez sur « Activer » pour activer la vérification en 2 étapes
![Activer la vérification en 2 étapes](/images/accounts/accounts-setting-up-multi-factor-authentication-mfa-for-your-account-155.png)
</Stepper>

## Inscription avec l'application d'authentification

Les utilisateurs peuvent s'inscrire en utilisant leur application d'authentification en scannant le code QR avec leur appareil mobile.

![Scanner le code QR avec l'application d'authentification](/images/accounts/accounts-setting-up-multi-factor-authentication-mfa-for-your-account-156.png)

Entrez le code à 6 chiffres affiché dans votre application d'authentification pour compléter la configuration de l'AMF.

![Entrer le code de vérification à 6 chiffres](/images/accounts/accounts-setting-up-multi-factor-authentication-mfa-for-your-account-157.png)

## Codes de sauvegarde pour une sécurité supplémentaire

Si vous perdez l'accès à votre appareil mobile, vous pouvez utiliser l'un des 10 codes de sécurité de sauvegarde. Ces codes servent de sécurité de secours, accordant l'accès quand votre appareil principal n'est pas disponible. Stockez ces codes de manière sécurisée.

## Indépendance pour personnaliser

Avec l'authentification multi-facteurs (AMF), chaque utilisateur de compte peut indépendamment activer ou désactiver l'AMF selon ses préférences et besoins.

## Dépannage

**Impossible de scanner le code QR**
- Assurez-vous que les permissions de caméra sont activées
- Augmentez la luminosité de l'écran
- Utilisez l'option de saisie manuelle si disponible
- Essayez une application d'authentification différente

**Le code à 6 chiffres ne fonctionne pas**
- Vérifiez que l'heure de votre appareil est synchronisée
- Entrez le code rapidement (expire en 30 secondes)
- Assurez-vous d'utiliser le bon compte dans l'application
- Essayez de générer un nouveau code

**Perte d'accès à l'application d'authentification**
- Utilisez l'un de vos 10 codes de sauvegarde
- Contactez le support avec la vérification du compte
- Ayez les informations de récupération du compte prêtes
- Considérez stocker les codes de sauvegarde de manière sécurisée

**Les codes de sauvegarde ne fonctionnent pas**
- Assurez-vous d'entrer le code correctement
- Chaque code de sauvegarde ne peut être utilisé qu'une seule fois
- Vérifiez si le code a déjà été utilisé
- Contactez le support si tous les codes échouent

**Souhaitez désactiver l'AMF**
- Allez aux paramètres de sécurité dans votre profil
- Cliquez sur « Désactiver » à côté de la vérification en 2 étapes
- Confirmez avec votre mot de passe
- Considérez les implications de sécurité avant de désactiver
