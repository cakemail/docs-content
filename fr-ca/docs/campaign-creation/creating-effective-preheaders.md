# Créer des pré-en-têtes efficaces

## Description

Un pré-en-tête ou texte d'aperçu est un court extrait de texte qui apparaît sous votre ligne d'objet dans de nombreux clients de courriel, tels que Outlook et Gmail. C'est un aperçu de l'information dans votre courriel.

## Pourquoi c'est important
Quand vous envoyez un courriel, une des premières choses que vos destinataires voient est le texte d'aperçu. En conséquence, il est important que cette première impression soit claire et invitante pour que les destinataires soient plus susceptibles d'ouvrir vos campagnes. Le fait est que vous devriez profiter de cette ressource ; elle peut être considérée comme le dernier coup pour impressionner vos lecteurs dans leur boîte de réception avant qu'ils fassent défiler, laissant votre campagne de courriel intacte.

### Processus étape par étape :

<Stepper>

1. Ouvrez le concepteur de courriel.

1. Cliquez sur Paramètres
   - Cherchez PROPRIÉTÉS OPTIONNELLES, puis détails du courriel.

1. Si vous avez du contenu et que votre infolettre est prête, vous pouvez cliquer sur la petite baguette magique pour générer des idées. 
  - Rechargez pour tester différentes idées, vous pouvez choisir le ton. - Gardez votre pré-en-tête court, moins de 130 caractères. Rendez-le complémentaire à votre ligne d'objet. 

1. Sauvegardez le pré-en-tête que vous voulez
</Stepper>

## Meilleures pratiques pour les pré-en-têtes

- **Gardez-le court** : Visez 35-90 caractères pour un affichage optimal
- **Complétez la ligne d'objet** : Ne la répétez pas, mais développez-la
- **Créez de l'urgence ou de la curiosité** : Encouragez les ouvertures avec un texte d'aperçu convaincant
- **Testez à travers les clients de courriel** : Différents clients affichent différentes longueurs
- **Utilisez un langage actionnable** : Incluez des verbes qui encouragent l'engagement
de votre courriel
- Certains clients de courriel peuvent ne pas supporter les pré-en-têtes personnalisés

## Exemples de bons pré-en-têtes

- **Objet** : « Nouveaux produits viennent d'arriver ! » **Pré-en-tête** : « Magasinez la dernière collection avant qu'elle soit épuisée »
- **Objet** : « Votre panier vous attend » **Pré-en-tête** : « Complétez votre achat et économisez 15% »
- **Objet** : « Infolettre hebdomadaire » **Pré-en-tête** : « Perspectives de l'industrie, conseils et offres exclusives à l'intérieur »

## Dépannage

**Pré-en-tête trop long**

- Gardez le texte sous 90 caractères pour la plupart des clients de courriel
- Testez les points de troncature dans différents clients
- Mettez les informations les plus importantes au début
- Utilisez les points de suspension (...) stratégiquement si nécessaire

**Problèmes de client de courriel**

- Certains clients de courriel plus anciens ne supportent pas les pré-en-têtes
- Gmail peut tirer du texte supplémentaire si le pré-en-tête est trop court
- Testez à travers les clients de courriel principaux avant d'envoyer
