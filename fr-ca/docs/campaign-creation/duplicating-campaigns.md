# Dupliquer des campagnes

## Description

Copier des campagnes précédentes est connu sous le nom de duplication. En sélectionnant dupliquer, vous copierez le gabarit, le contenu et la liste de contacts que vous avez originalement sélectionnés. Les campagnes passées sont dupliquées pour différentes raisons. Habituellement, elles sont répliquées pour économiser du temps et réduire le risque d'erreurs quand vous renvoyez le même gabarit à une liste.

## Pourquoi c'est important
Il est souvent très utile de dupliquer votre campagne pour plusieurs raisons. Par exemple, si vous avez aimé la structure de votre campagne et aimeriez la réutiliser dans le futur, dupliquez-la simplement ! Des copies peuvent également être créées pour tester des changements à une structure de campagne existante.

## Instructions étape par étape.

<Stepper>

1. Sélectionnez l'option **Campagnes** dans le menu du haut
   ![Sélectionner le menu Campagnes](/images/campaign-creation/3D2ABF07-0266-46AA-B71A-D79E05D7A89C_4_5005_c.jpeg)

1. Pour accéder à vos brouillons ou campagnes passées, sélectionnez une des quatre options : 
   - Brouillons
   - Boîte d'envoi
   - Envoyées
   - Gabarits
   ![Sélectionner le type de campagne](/images/campaign-creation/5EE90D39-7AF4-4948-87E1-3B3C730DE3EB_4_5005_c.jpeg)

1. Sélectionnez la campagne désirée.

1. Cliquez sur les **trois points** à droite de votre onglet de campagne désiré. 
   Pour une campagne qui a déjà été envoyée, cliquez simplement sur **Dupliquer**
   ![Cliquer sur le menu trois points](/images/campaign-creation/252616FF-F3F1-4A08-A7D9-0EE560BCCC3F_4_5005_c.jpeg)

1. Sélectionnez **Dupliquer**
   ![Sélectionner l'option Dupliquer](/images/campaign-creation/79BB9E09-3455-489A-81B5-F71621042E08_4_5005_c.jpeg)
</Stepper>

#### La duplication est terminée
Une fois que la copie est complète, vous pouvez changer la liste de contacts, les informations de campagne, et le contenu selon les besoins.

## Dépannage

**La duplication prend trop de temps**

- Les grandes campagnes avec plusieurs images peuvent prendre du temps
- Vérifiez la stabilité de votre connexion Internet
- Essayez de dupliquer pendant les heures creuses

**Contenu manquant après duplication**

- Vérifiez que toutes les images ont été correctement téléchargées dans la bibliothèque
- Vérifiez que le HTML/CSS personnalisé est compatible
- Assurez-vous que les sources de contenu dynamique sont encore actives

**La liste de contacts ne se copie pas**

- Confirmez que la liste originale existe encore
- Vérifiez que vous avez accès à la liste de contacts
- Sélectionnez manuellement la liste après duplication si nécessaire

**Les changements de gabarit ne se reflètent pas**

- Videz le cache du navigateur après duplication
- Sauvegardez la campagne dupliquée avant de l'éditer
- Vérifiez que vous éditez la copie, pas l'original
