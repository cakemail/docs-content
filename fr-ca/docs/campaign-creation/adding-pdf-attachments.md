# Ajouter des pièces jointes PDF

## Pièces jointes dans les courriels

Il y a plusieurs raisons pour lesquelles vous ne devriez pas envoyer de courriels avec des pièces jointes : certains FAI peuvent les marquer comme pourriel, les abonnés peuvent croire que c'est du pourriel, et le message lui-même sera si volumineux qu'il devient un tracas pour le destinataire. Nous avons une meilleure alternative pour vous, une qui est plus sûre et plus efficace.

## Joindre des fichiers correctement

Si vous devez diriger du contenu vers votre message afin de surmonter les obstacles mentionnés ci-dessus, nous pouvons héberger votre fichier en ligne, sur notre gestionnaire de fichiers, et ajouter un lien vers votre contenu de courriel. Vous améliorerez votre délivrabilité et améliorerez l'expérience utilisateur pour vos destinataires.

## Télécharger votre PDF

D'abord, importez le PDF dans la bibliothèque, tout comme vous le feriez pour une image. Voici comment procéder :

<Stepper>

1. Ouvrez la campagne.
![DD222A1A-C2FC-4946-97D9-77F6BB8BD302_4_5005_c.jpeg](/images/campaign-creation/campaigns-adding-a-pdf-to-a-campaign-270.jpeg)

1. Dans le contenu du courriel, glissez et déposez un bouton.
![Image](/images/campaign-creation/mceclip0.png)

1. Cliquez sur le bouton, et dans la colonne de droite sous « Action », cliquez sur « Lier un fichier »
![Bouton de lien de fichier dans le menu d'action](/images/campaign-creation/pdf.png)

1. Cliquez sur Télécharger et sortir. (Les PDF ne sont pas visibles dans la bibliothèque)
![Dialogue de téléchargement pour les fichiers PDF](/images/campaign-creation/campaigns-adding-a-pdf-to-a-campaign-273.jpeg)

![Screen_Shot_2021-07-05_at_2.18.49_PM.png](/images/campaign-creation/campaigns-adding-a-pdf-to-a-campaign-274.png)

   Sauvegardez et voilà !

</Stepper>

## Dépannage

**L'échec du téléchargement PDF**

- Vérifiez que la taille du fichier est sous la limite maximale (typiquement 10 Mo)
- Assurez-vous que le PDF n'est pas corrompu ou protégé par mot de passe
- Essayez de convertir au format PDF/A pour une meilleure compatibilité

**Le lien ne fonctionne pas dans le courriel**

- Vérifiez que le PDF a été téléchargé avec succès dans le gestionnaire de fichiers
- Vérifiez que l'URL du lien est complète et correctement formatée
- Testez le lien en mode aperçu avant d'envoyer

**Stockage du gestionnaire de fichiers plein**

- Supprimez les anciens fichiers ou les fichiers inutilisés du gestionnaire de fichiers
- Compressez les PDF avant de les télécharger pour économiser l'espace
- Considérez mettre à niveau votre plan pour plus de capacité de stockage
