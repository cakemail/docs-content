# Tester votre campagne

## Description

Avant d'envoyer votre campagne, vous pouvez la tester correctement en la prévisualisant. Nous vous montrerons comment le faire et ce que vous devez vérifier avant de l'envoyer. Avant de cliquer avec confiance sur votre bouton d'envoi, nous vous guiderons à travers toutes les questions auxquelles vous devez répondre.

## Pourquoi c'est important

Pour vous assurer d'envoyer la meilleure version de votre courriel, vous devez le tester minutieusement avant qu'il soit envoyé. Différents navigateurs et plateformes peuvent afficher votre contenu différemment, affectant l'expérience du destinataire. Aussi : n'oubliez pas de tester CHAQUE LIEN !

## Instructions étape par étape

<Stepper>

1. Cliquez sur l'**icône œil** sur le côté supérieur droit du concepteur de courriel. 
Vous pouvez voir le contenu du courriel soit sur la version bureau ou la version mobile.
![Bouton d'aperçu avec icône œil dans le concepteur de courriel](/images/campaign-creation/377D4626-1C72-482D-BC74-E00B118EA588_4_5005_c.jpeg)

1. Vérifiez le mode sombre au même endroit.

1. Envoyez un courriel de test
Testez votre courriel (vous pourriez aussi vouloir l'envoyer à un ami pour vérifier les fautes de frappe), pour que vous puissiez voir comment vos destinataires le verront. Assurez-vous de vous envoyer un test de votre courriel en cliquant sur la flèche dans le coin supérieur droit du concepteur de courriel et en sélectionnant **Envoyer un courriel de test**.
![Option d'envoi de courriel de test dans le menu déroulant](/images/campaign-creation/A3D362A1-BA3E-4527-9506-8BD2EB687E18_4_5005_c.jpeg)
En conséquence, chaque fois que vous changez quelque chose, envoyez-vous un autre test. En général, tout doit être vérifié deux fois.
</Stepper>

## Liste de vérification des choses à vérifier avant d'envoyer
Voici quelques points à considérer, en regardant dans le concepteur de courriel.

#### Aperçu
☐ Vérifiez les versions bureau et mobile pour vous assurer que l'information s'affiche approprié.

#### Pré-en-tête
☐ Y a-t-il un lien pour montrer le courriel dans le navigateur ? 

☐ Est-il expliqué à l'abonné qu'il s'est abonné à l'infolettre ?

#### Le corps du courriel
☐ Toutes les images s'affichent-elles correctement ? 

☐ Quand vous passez la souris sur les images, y a-t-il un texte alternatif clair qui apparaît ? 

☐ Tous les liens fonctionnent-ils ? 

☐ Les liens vous amènent-ils au bon endroit ? 

☐ Les caractères spéciaux s'affichent-ils correctement ?

#### Pied de page
☐ Avez-vous un lien de désinscription `[UNSUBSCRIBE]` ? ☐ Avez-vous la balise `[CLIENT.ADDRESS]` ?

### Envoyez votre test

Vous pouvez continuer à l'étape suivante quand vous avez fini de tester en cliquant sur le bouton **Continuer**.

![Bouton Continuer pour procéder au résumé de campagne](/images/campaign-creation/2E25B84B-4E02-4F2B-922E-386FD0E58407_4_5005_c.jpeg)

Ici il serait utile si vous vérifiez trois fois le résumé de la campagne et choisissez les destinataires :

☐ **Nom de campagne :** Choisissez le nom de la campagne  

☐ **Nom et courriel de l'expéditeur :** Assurez-vous que l'expéditeur est reconnaissable  

☐ **Pré-en-tête :** clair, attrayant et utile

☐ **À :** Choisissez la liste correcte  

☐ **Objet :** L'objet devrait être clair et ne contenir aucune faute de frappe

### Choisissez la date et l'heure d'envoi

#### Envoyer dès que possible
Si vous êtes prêt à l'envoyer maintenant, sélectionnez l'option Envoyer dès que possible et cliquez sur le bouton Envoyer.

#### Programmer pour plus tard
Sélectionnez l'option Spécifier une heure d'envoi et entrez une date et une heure pour programmer votre courriel pour plus tard. La campagne peut être conçue deux années civiles à l'avance (l'année actuelle et l'année suivante). 

### Envoyez votre campagne

![Bouton d'envoi de campagne](/images/campaign-creation/E14FA32F-8738-485D-8216-455565EE0491.png)

En suivant ce guide, vous serez prêt à cliquer sur ce bouton vert d'envoi avec confiance.

## Dépannage

**Courriel de test non reçu**

- Vérifiez les dossiers pourriel/indésirable pour les courriels de test
- Vérifiez que l'adresse courriel de test est correcte
- Accordez 5-10 minutes pour les délais de livraison

**L'aperçu ne se charge pas**

- Videz le cache et les cookies du navigateur
- Essayez un navigateur différent (Chrome/Firefox recommandé)
- Désactivez temporairement les extensions de navigateur

**Images brisées dans le test**

- Assurez-vous que les images sont téléchargées dans la bibliothèque d'abord
- Vérifiez que les URL d'images utilisent HTTPS
- Vérifiez que les tailles de fichiers d'images sont sous 1 Mo chacune

**Liens non cliquables**

- Confirmez que les liens incluent l'URL complète (https://...)
- Testez les liens en dehors de l'éditeur de courriel d'abord
- Vérifiez les caractères spéciaux qui brisent l'URL

**Problèmes d'aperçu mobile**

- Utilisez des tests d'appareils réels pour la précision
- Vérifiez les points de rupture du design réactif
- Testez avec plusieurs tailles d'écran

**Les balises de fusion ne s'affichent pas**

- Les balises de fusion ne montreront pas les données réelles dans les aperçus
- Envoyez un courriel de test pour voir la personnalisation
- Vérifiez que la syntaxe de balise de fusion est correcte
