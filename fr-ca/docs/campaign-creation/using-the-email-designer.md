# Utiliser le concepteur de courriel

<iframe src="//www.youtube-nocookie.com/embed/az96wcaWOnA" width="560" height="315" frameborder="0" allowfullscreen=""></iframe>

## Description

C'est l'outil d'édition principal que vous utiliserez pour créer un gabarit de courriel ; il est facile à utiliser et permet l'ajout et la suppression rapides de sections, la définition de différentes couleurs pour différentes zones, la décision sur la police et la largeur de votre campagne, les tests et bien plus, tout depuis une seule fenêtre.

## Pourquoi c'est utile

Cette fonction d'édition permet aux utilisateurs de créer des campagnes de courriel accrocheuses sans écrire de code pour accomplir n'importe quel objectif désiré. Avec cet outil, n'importe qui peut faire un excellent courriel. Le concepteur de courriel est maintenant plus flexible avec le déplacement de sections, la réorganisation de contenu, la fonctionnalité de sélecteur de couleur, et le versioning de contenu. Ici nous explorerons tout ce qui concerne notre concepteur.

## Comment utiliser le concepteur de courriel

Voici quelques fonctionnalités de cet outil pour commencer et l'utiliser à son plein potentiel.

### Paramètres

Cette partie est pour définir le ton pour toute la campagne. Il y a beaucoup d'options à considérer :

- Largeur de zone de contenu (nous recommandons de ne pas dépasser 600 pixels)

- Alignement de zone de contenu. (Gauche ou Centre)

- Couleur d'arrière-plan

- Couleur d'arrière-plan de zone de contenu

- Police par défaut

- Couleur de lien

![Panneau de paramètres du concepteur de courriel montrant les couleurs d'arrière-plan, polices, et options de mise en page](/images/campaign-creation/campaigns-using-the-email-designer-312.png)

#### Ajoutez votre pré-en-tête

Pour tirer le maximum de votre campagne, n'oubliez pas d'ajouter un pré-en-tête convaincant. Faites simplement défiler vers le bas dans vos paramètres vers vos Propriétés optionnelles, et entrez-le dans le champ pré-en-tête. C'est aussi simple que ça !

![Champ de texte pré-en-tête dans la section propriétés optionnelles](/images/campaign-creation/campaigns-using-the-email-designer-313.png)

Si vous avez besoin d'un peu d'inspiration, notre fonctionnalité IA est là pour vous aider à générer des pré-en-têtes engageants qui résonnent avec votre audience. Essayez-la et trouvez l'ajustement parfait pour votre message.

![Interface de génération de pré-en-tête alimentée par l'IA](/images/campaign-creation/campaigns-using-the-email-designer-314.png)

### Rangées

Avant d'ajouter du contenu à votre campagne, vous devez choisir une rangée. La section Rangées consiste en une variété de structures de composant. Veuillez sélectionner celle que vous voulez et la glisser n'importe où dans la campagne. Testez-les et trouvez celle qui fonctionne le mieux pour votre gabarit.

![Options de mise en page de rangée montrant diverses configurations de colonnes](/images/campaign-creation/campaigns-using-the-email-designer-315.png)

![Glisser et déposer une mise en page de rangée dans le canevas de courriel](/images/campaign-creation/campaigns-using-the-email-designer-316.png)

#### Déplacer / Supprimer / Dupliquer la rangée

Repositionnez, supprimez ou dupliquez une section du courriel :

- D'abord, sélectionnez la rangée que vous devez repositionner pour accéder à son menu de repositionnement.

Pour déplacer une section vers le haut ou le bas dans le gabarit :

- Dans le menu de positionnement, cliquez sur les flèches croisées trouvées à gauche pour déplacer votre rangée sélectionnée.

![Contrôles de positionnement de rangée avec flèches croisées pour déplacer les sections](/images/campaign-creation/campaigns-using-the-email-designer-317.png)

Pour supprimer une section :

- Dans le menu de positionnement, cliquez sur l'icône de poubelle à droite pour supprimer la rangée.

![Bouton de suppression de rangée avec icône de poubelle](/images/campaign-creation/campaigns-using-the-email-designer-318.png)

Pour dupliquer la rangée :

- Dans le menu de positionnement, cliquez sur le bouton de fenêtre double à droite, à côté de l'icône de poubelle.

![Bouton de duplication de rangée avec icône de fenêtre double](/images/campaign-creation/campaigns-using-the-email-designer-319.png)

#### Modifier

Plusieurs éléments peuvent être modifiés dans la section rangée. Sélectionnez la rangée que vous souhaitez modifier pour accéder à ce menu sur la partie droite du concepteur.

Voici les fonctionnalités que vous pouvez contrôler :

- Couleur d'arrière-plan de rangée

- Couleur d'arrière-plan de contenu

- Ne pas empiler sur mobile

- Image d'arrière-plan de rangée

- Personnaliser la colonne

- Condition d'affichage

- Cacher sur Bureau/Mobile

- Personnaliser les colonnes

- Arrière-plan de colonne

- Rembourrage

- Bordures

![Panneau de personnalisation de rangée montrant les paramètres de colonne, rembourrage, et bordures](/images/campaign-creation/campaigns-using-the-email-designer-320.png)

### Contenu

Ayant décidé quel type de rangées vous voulez, vous pouvez maintenant choisir quel contenu inclure dans ces rangées.

Les options de contenu suivantes sont disponibles :

![Palette de blocs de contenu montrant tous les éléments disponibles comme Titre, Paragraphe, Image, Bouton, etc.](/images/campaign-creation/campaigns-using-the-email-designer-321.png)

- **Titre** : Utilisez ceci pour insérer le titre de votre campagne.

- **Paragraphe** : Utilisez ceci pour insérer un paragraphe de texte.

- **Liste** : Utilisez ceci pour insérer une liste de contenu différent.

- **Image** : Utilisez ceci pour insérer une image que vous avez téléversée ou une image stock.

- **Bouton** : Utilisez ceci pour insérer un bouton d'appel à l'action visible.

- **Diviseur** : Utilisez ceci pour diviser votre contenu.

- **Espacement** : Ce bloc peut insérer de l'espace dans votre contenu de courriel.

- **Social** : Insérez les icônes de médias sociaux et liez-les à vos pages.

- **HTML** : Insérez votre code HTML personnalisé dans l'éditeur avec cette option.

- **Vidéo** : Insérez une vignette montrant un aperçu de votre vidéo et un lien vers celle-ci.

- **Menu** : Utilisez ceci pour inclure un menu interactif dans votre contenu de courriel.

- **GIFS** : Utilisez ceci pour inclure un gif animé dans votre courriel et choisir parmi diverses options.

Chacun d'eux est modifiable dans son bloc. Glissez et déposez l'élément désiré dans votre section de contenu de courriel pour les utiliser. Pour créer la campagne parfaite, expérimentez avec toutes les options dans le concepteur. Il n'y a presque aucune limite à ce que cet outil peut faire !

#### Revenir à une version précédente d'une campagne

Une version antérieure de votre courriel peut être restaurée. Dans le concepteur, cliquez sur l'icône « horloge » en bas à gauche de votre campagne. Vous pouvez alors revenir à une action spécifique dans l'édition.

![Icône d'horloge d'historique de version dans le coin inférieur gauche](/images/campaign-creation/campaigns-using-the-email-designer-322.png)

![Panneau d'historique de version montrant la liste des versions sauvegardées avec horodatages](/images/campaign-creation/campaigns-using-the-email-designer-323.png)

#### Mode aperçu

Vous pouvez prévisualiser le site en mode bureau et mobile en cliquant sur le bouton icône Œil trouvé en haut à droite du concepteur. Testez votre contenu sur les deux plateformes et assurez-vous qu'il a l'air bien.

![Bouton de mode aperçu avec icône œil pour les vues bureau et mobile](/images/campaign-creation/campaigns-using-the-email-designer-324.png)

#### Sauvegarder les changements à la campagne.

![Bouton vert de sauvegarde pour préserver les changements de campagne](/images/campaign-creation/campaigns-using-the-email-designer-325.png)

Cliquez toujours sur le bouton vert Sauvegarder lors de la modification de votre campagne dans le concepteur de courriel. Il vaut mieux être prudent que désolé !

#### Sauvegarder la campagne comme gabarit, la réinitialiser ou l'abandonner

Si vous voulez choisir une de ces options, cliquez sur le bouton flèche en haut à droite de l'écran. Vous pouvez en apprendre plus sur la sauvegarde de gabarits ici.

![Menu déroulant d'options supplémentaires avec Sauvegarder comme gabarit, Réinitialiser, et options d'abandon](/images/campaign-creation/campaigns-using-the-email-designer-326.png)

#### Détails de campagne

Si vous êtes satisfait de votre gabarit et souhaitez l'envoyer, cliquez sur Continuer dans le coin supérieur droit du concepteur. La section Détails de campagne vous fournit ces options pour envoyer la campagne.

Vous pouvez définir ce qui suit ici :

- Le nom de votre campagne

- La liste des abonnés

- Le segment dans cette audience,

- Le nom de l'expéditeur

- L'adresse courriel de l'expéditeur

- La ligne d'objet

Vous pouvez également revoir la campagne avec un test de courriel et programmer la livraison en sélectionnant Date et heure d'envoi.

![Formulaire de détails de campagne montrant les champs pour nom, liste, segment, info expéditeur, et ligne d'objet](/images/campaign-creation/campaigns-using-the-email-designer-327.png)

Rappelez-vous de la tester minutieusement avant de l'envoyer par courriel. Voici une liste de vérification de choses à faire avant d'envoyer une campagne et après qu'elle ait été envoyée.

## Dépannage

**Le concepteur de courriel ne se charge pas**

- Videz le cache et les témoins du navigateur
- Utilisez Chrome ou Firefox pour la meilleure compatibilité
- Désactivez les bloqueurs de publicité et extensions de navigateur

**Les changements ne se sauvegardent pas**

- Cliquez fréquemment sur le bouton vert Sauvegarder
- Vérifiez la stabilité de la connexion Internet
- Cherchez les messages de confirmation de sauvegarde automatique

**Problèmes d'aperçu mobile**

- Utilisez des tests d'appareils réels pour la précision
- Vérifiez les paramètres « Ne pas empiler sur mobile »
- Ajustez les largeurs de colonne pour l'affichage mobile

**Les images ne s'insèrent pas**

- Téléversez d'abord les images dans la bibliothèque
- Vérifiez la taille du fichier image (max 1 Mo)
- Vérifiez les dimensions d'image (max 1600x2000px)

**Le HTML personnalisé ne fonctionne pas**

- Validez le code HTML avant de l'insérer
- Supprimez JavaScript (non supporté dans les courriels)
- Utilisez du CSS en ligne au lieu de feuilles de style externes

**Problèmes d'historique de version**

- L'historique de version est limité aux modifications récentes
- Sauvegardez les versions importantes comme gabarits
- Ne peut pas récupérer les versions de plus de 30 jours
