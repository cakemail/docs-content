# Ajouter votre adresse physique

## Description

Dans le cadre de notre politique anti-pourriel, une adresse physique doit être incluse dans votre courriel. Voici quelques directives sur le formatage de l'adresse physique trouvée au bas du courriel. Il y a deux façons de le faire :

- Ajouter votre adresse physique dans le concepteur de courriel
- Ajouter votre adresse physique dans l'éditeur HTML

## Pourquoi c'est important

Légalement, cette adresse doit être fournie à des fins de responsabilité afin d'identifier l'expéditeur au destinataire. Cela peut également aider à faciliter le processus de désinscription, prévenir les plaintes et aider à améliorer la livraison.

## Exigences légales

Veuillez noter que dans l'opinion de la Commission, « adresse postale » peut être l'adresse valide et actuelle de rue (ou civique) de l'expéditeur, l'adresse de boîte postale, l'adresse de route rurale, ou l'adresse de livraison générale et doit être valide pour un minimum de **60 jours après que le message ait été envoyé**.

## Instructions étape par étape
### Modifier votre adresse physique dans votre profil

Une adresse physique doit être incluse pour programmer une infolettre afin de protéger les abonnés (et vous-même). Si le système ne peut pas détecter le champ de fusion de l'adresse dans la campagne, il l'ajoutera automatiquement au bas. De plus, entrer les informations de votre entreprise est requis quand votre compte est créé.

<Stepper>

1. Sélectionnez Compte dans le menu en haut à droite
![Bouton Sélectionner modifier le courriel](/images/campaign-creation/485F26F5-0D6B-4670-B865-2430AEAFD27E_4_5005_c.jpeg)

1. Sélectionnez l'onglet Profil dans la page Compte
![Coller le code HTML](/images/campaign-creation/FCAD0D54-BEF5-46E5-9058-B8E4A751B4AA_4_5005_c.jpeg)

1. Dans les options de Profil, cliquez sur l'icône Crayon trouvée à droite d'Adresse (Requis)
![Sélectionner Commencer avec votre propre code](/images/campaign-creation/DDD01F59-144A-4FB6-A2BA-7DE791677C9B_4_5005_c.jpeg)

1. Mettez à jour votre adresse si nécessaire et cliquez sur sauvegarder.
</Stepper>








### Ajouter votre adresse physique dans le concepteur de courriel
Vous pouvez ajouter [CLIENTS.ADDRESS] où vous voulez que l'adresse apparaisse dans votre campagne. L'adresse de votre entreprise sera tirée de la page de profil de l'entreprise. Il sera ensuite possible de modifier le champ [CLIENTS.ADDRESS] avec la police, la couleur et la taille.

<Stepper>

1. Sélectionnez Campagnes.
![Sélectionner le gabarit de campagne](/images/campaign-creation/3D2ABF07-0266-46AA-B71A-D79E05D7A89C_4_5005_c.jpeg)

1. Sélectionnez votre gabarit de courriel de campagne.
![Formulaire de mise à jour d'adresse](/images/campaign-creation/0E44E1D7-F230-4C01-8183-F606B426F1CF_4_5005_c.jpeg)

1. Sélectionnez Modifier le courriel.
![Menu d'utilisation de balises de fusion](/images/campaign-creation/63842824-8F4D-4E01-83AA-021FD47A1891_4_5005_c.jpeg)

1. Tapez [CLIENTS.ADDRESS] où vous voulez que votre adresse physique apparaisse
Personnalisez-la avec la couleur de texte, la taille, etc.
![Ajouter CLIENTS.ADDRESS dans l'éditeur HTML](/images/campaign-creation/A3405E2E-7DA4-4E95-B028-D9DF1B06417B_4_5005_c.jpeg)

1. Alternativement, vous pouvez ajouter le champ d'adresse physique avec l'option Balises de fusion et en sélectionnant Adresse complète.
![](/images/campaign-creation/Merge%20tags%20and%20clients%20address.gif)
Vous pourrez voir le résultat sur la page suivante, où vous pouvez programmer la campagne.
</Stepper>







### Ajouter votre adresse physique dans l'éditeur HTML  
En sélectionnant l'option « commencer à partir de zéro », vous pouvez créer votre gabarit à partir de votre propre code HTML. Pour ajouter correctement l'adresse physique avec cette option, suivez ces étapes :

<Stepper>

1. Sélectionnez Créer dans le menu du haut.
![Taper la balise de fusion CLIENTS.ADDRESS](/images/campaign-creation/6208D820-32F4-4547-B8CA-5D9F75C2FE29_4_5005_c.jpeg)

1. Cliquez sur « Campagne » dans le nouveau menu qui apparaît.
![Sélectionner le menu Compte](/images/campaign-creation/7D56F11C-29E9-40E3-9D83-03F10FF6AC93_4_5005_c.jpeg)

1. Sélectionnez l'onglet Commencer à partir de zéro.
![Cliquer sur l'option Campagne](/images/campaign-creation/2234E183-9EA0-4A50-BC6D-5759A1CB5159_4_5005_c.jpeg)

1. Cliquez sur Sélectionner sous l'option Commencer avec votre propre code.
![Sélectionner l'onglet Profil](/images/campaign-creation/37BA470F-8AD5-4FBF-A1A3-DC4257B920E8.png)

1. Collez/écrivez votre code sur la partie gauche de la page.

1. En haut à droite, cliquez sur Télécharger et modifier.
![Sélectionner l'onglet Commencer à partir de zéro](/images/campaign-creation/354BF584-D33E-4725-B566-F5CCE03BCEF2.png)

1. Tapez [CLIENTS.ADDRESS] où vous voulez que votre adresse physique apparaisse et personnalisez-la avec la couleur de texte, la taille, etc.

1. Surlignez le champ d'adresse physique.
![Naviguer vers la section Campagnes](/images/campaign-creation/7DF5A948-7E0C-431F-9673-B2355099823E_4_5005_c.jpeg)

1. Vous pouvez modifier le champ d'adresse physique avec le menu du haut.
![Cliquer sur l'icône crayon pour modifier l'adresse](/images/campaign-creation/1118ADB0-2764-4B83-8E65-A33F8A542F2B_4_5005_c.jpeg)
</Stepper>

## Exigences d'adresse

Votre adresse physique doit être :
- **Valide et actuelle** pendant au moins 60 jours après l'envoi
- **Complète** avec adresse de rue ou boîte postale
- **Accessible** à des fins de responsabilité
- **Légalement conforme** aux réglementations anti-pourriel

## Types d'adresses acceptables

- Adresse de rue (civique) valide et actuelle
- Adresse de boîte postale  
- Adresse de route rurale
- Adresse de livraison générale

## Avantages d'inclure l'adresse physique

- **Conformité légale** : Répond aux exigences des lois anti-pourriel
- **Renforcement de la confiance** : Montre la transparence et la légitimité
- **Amélioration de la livraison** : Aide avec la délivrabilité des courriels
- **Prévention des plaintes** : Fournit une identification claire de l'expéditeur
- **Apparence professionnelle** : Démontre une présence d'entreprise établie

## Meilleures pratiques

- Gardez vos informations d'adresse à jour dans votre profil
- Utilisez un formatage cohérent à travers toutes les campagnes
- Placez les informations d'adresse dans le pied de page du courriel pour la visibilité
- Assurez-vous que l'adresse reste valide pour la période requise

## Dépannage

**L'adresse n'apparaît pas dans le courriel**

- Vérifiez si le champ de fusion d'adresse est correctement inséré
- Vérifiez que votre profil contient une adresse physique complète
- Prévisualisez le courriel pour confirmer l'ajout automatique au bas

**La campagne ne s'envoie pas**

- Assurez-vous qu'une adresse physique valide est incluse
- Mettez à jour l'adresse de votre profil si elle manque ou est incomplète
- Vérifiez que l'adresse répond aux exigences légales (pas seulement courriel/web)

**Problèmes de formatage d'adresse**

- Utilisez le formatage d'adresse standard pour votre pays
- Évitez les caractères spéciaux qui peuvent ne pas s'afficher correctement
- Testez comment l'adresse apparaît dans différents clients de courriel

**L'adresse de profil ne se met pas à jour**

- Sauvegardez les changements après avoir mis à jour les informations de votre profil
- Déconnectez-vous et reconnectez-vous pour actualiser les données en cache
- Contactez le support si les changements ne persistent pas

**Avertissements de conformité**

- Vérifiez que l'adresse restera valide pour 60+ jours
- N'utilisez pas d'adresses temporaires ou de bureaux virtuels sans présence physique
- Assurez-vous que les adresses de boîte postale sont correctement enregistrées et actives
