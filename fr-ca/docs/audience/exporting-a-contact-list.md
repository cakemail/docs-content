# Exporter une liste de contacts

## Description

Vous pouvez exporter votre liste de contacts sous forme de fichier .csv pour l'utiliser dans un outil externe. Cette exportation inclura l'ID de l'abonné, le statut de l'adresse, l'adresse courriel et d'autres attributs que vous avez configurés. Les adresses exportées ne seront pas supprimées de la liste de contacts.

## Exigences du plan

**Note :** L'exportation d'une liste est disponible exclusivement aux utilisateurs abonnés à nos **plans Croissance ou Premium**. Débloquez cette fonctionnalité pour télécharger facilement vos données et avoir un meilleur contrôle sur vos contacts.

## Pourquoi c'est important

Exporter votre liste de contacts peut être un moyen utile de sauvegarder vos données, d'analyser votre audience ou de transférer des contacts vers une autre plateforme. Si vous voulez gérer vos listes en dehors de notre application ou vous y référer pour faire des recoupements, cette fonction peut être utile. Lorsque vous lancez une exportation de liste de contacts, le fichier restera dans les paramètres de votre compte pendant cinq jours.

## Ce qui est inclus dans l'exportation

Le fichier CSV exporté inclura :
- **ID de l'abonné**
- **Statut de l'adresse** (actif, désabonné, etc.)
- **Adresse courriel**
- **Attributs personnalisés** que vous avez configurés
- **Date d'abonnement** et autres métadonnées

## Notes importantes

- Les adresses exportées restent dans votre liste de contacts originale
- Les fichiers d'exportation sont disponibles pendant **5 jours** après la génération
- Disponible uniquement sur les plans Croissance et Premium
- Utile pour la sauvegarde de données, l'analyse ou la migration de plateforme

## Dépannage

**Option d'exportation non disponible**

- Vérifiez que vous avez un plan Croissance ou Premium
- Vérifiez les permissions de votre compte
- Assurez-vous que la liste contient des contacts à exporter
- Essayez de sélectionner la liste d'abord avant d'accéder aux Actions

**L'exportation prend trop de temps à traiter**

- Les grandes listes peuvent prendre 10-30 minutes
- Vérifiez votre courriel pour la notification de fin
- Essayez d'exporter un segment plus petit d'abord
- Assurez-vous que votre courriel de compte est valide

**Impossible de télécharger le fichier exporté**

- Rappelez-vous que les fichiers expirent après 5 jours
- Vérifiez le dossier pourriel pour le lien de téléchargement
- Assurez-vous que le navigateur autorise les téléchargements
- Essayez un navigateur différent ou désactivez les extensions

**Le fichier CSV semble corrompu**

- Ouvrez avec un programme de tableur, pas un éditeur de texte
- Vérifiez les paramètres de délimiteur CSV (virgule vs point-virgule)
- Essayez d'importer dans Google Sheets d'abord
- Assurez-vous que les caractères spéciaux sont encodés en UTF-8

**Données manquantes dans l'exportation**

- Vérifiez que les attributs ont été correctement configurés
- Vérifiez les critères du segment si vous exportez un segment
- Assurez-vous que les données existent pour les champs exportés
- Essayez d'exporter la liste complète au lieu du segment

**L'exportation inclut des contacts non désirés**

- Révisez attentivement les critères du segment
- Vérifiez les filtres de statut des contacts
- Assurez-vous d'avoir sélectionné la bonne liste
- Envisagez de créer un segment plus spécifique
