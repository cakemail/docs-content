# Ajouter des attributs à votre liste

## Description

Les attributs vous permettent de collecter des informations supplémentaires sur vos contacts au-delà de leurs noms et adresses courriel. Ces points de données supplémentaires vous aident à personnaliser vos campagnes de courriel et à créer des segments plus ciblés. Les attributs peuvent inclure des éléments comme les anniversaires, les préférences, l'emplacement, ou toute autre information pertinente pour votre entreprise.

## Pourquoi c'est important

L'ajout d'attributs à vos listes de contacts améliore considérablement votre capacité à personnaliser et segmenter vos campagnes de courriel. Avec des attributs, vous pouvez créer du contenu plus pertinent qui résonne avec des groupes spécifiques dans votre audience, conduisant à de meilleurs taux d'engagement et conversions. Les attributs vous permettent également d'automatiser les campagnes basées sur des déclencheurs spécifiques et de suivre les informations importantes du parcours client qui peuvent informer vos stratégies marketing futures.

1. Configurez votre attribut
**Nom de l'attribut :** Donnez un nom descriptif à votre attribut
**Type de données :** Choisissez parmi :
- Texte (pour des réponses courtes)

- Date (pour les anniversaires, dates d'anniversaire, etc.)

## Types d'attributs et exemples d'utilisation

**Attributs texte :**
- Nom de l'entreprise
- Titre du poste
- Ville
- Préférences de produit

**Attributs numériques :**
- Âge
- Revenu annuel
- Nombre d'achats
- Score de fidélité

**Attributs de date :**
- Date de naissance
- Date d'anniversaire de l'entreprise
- Date du dernier achat
- Date d'abonnement

**Attributs Vrai/Faux :**
- Accepte les courriels promotionnels
- Client VIP
- A acheté ce mois-ci
- Intéressé par les mises à jour de produit

## Dépannage

**Impossible de créer un nouvel attribut**

- Vérifiez que vous avez les permissions d'administrateur
- Assurez-vous que le nom de l'attribut n'existe pas déjà
- Évitez les caractères spéciaux dans les noms d'attributs
- Vérifiez que vous n'avez pas atteint la limite d'attributs de votre plan

**L'attribut n'apparaît pas dans les formulaires**

- Actualisez la page après avoir créé l'attribut
- Vérifiez que l'attribut a été enregistré correctement
- Assurez-vous que l'attribut est associé à la bonne liste
- Vériez la configuration du formulaire d'abonnement

**Les données d'attribut ne s'importent pas correctement**

- Vérifiez que les en-têtes de colonne correspondent exactement aux noms d'attributs
- Assurez-vous que le format de données correspond au type d'attribut
- Utilisez le format AAAA-MM-JJ pour les dates
- Vérifiez les espaces ou caractères supplémentaires dans les données

**Impossible de modifier un attribut existant**

- Notez que certains types d'attributs ne peuvent pas être modifiés après création
- Essayez de créer un nouvel attribut avec les paramètres corrects
- Exportez les données avant de supprimer et recréer l'attribut
- Vérifiez que l'attribut n'est pas utilisé dans des campagnes actives

**Les attributs ne fonctionnent pas dans la segmentation**

- Assurez-vous que les contacts ont des données pour cet attribut
- Vérifiez que les critères de segment utilisent la syntaxe correcte
- Accordez du temps pour que les nouvelles données d'attribut se traitent
- Testez avec un segment simple d'abord

**Les attributs de date ne fonctionnent pas correctement**

- Utilisez le format AAAA-MM-JJ pour toutes les dates
- Vérifiez que les dates sont dans une plage valide
- Assurez-vous qu'aucun champ de date n'est laissé vide à moins d'être optionnel
- Testez avec une date d'échantillon d'abord
