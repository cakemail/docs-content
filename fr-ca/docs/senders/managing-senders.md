# Gérer les expéditeurs

## Description

Dans cet article, nous discuterons comment choisir votre nom d'expéditeur et adresse courriel tout en restant cohérent avec votre marque, ce qui affectera directement la livraison de vos campagnes et votre image. En conséquence, chaque message inclut les adresses de l'expéditeur et du destinataire.

## Pourquoi c'est important

Le serveur de courriel d'envoi voit le domaine « expéditeur » lors de l'initiation de la session et vos destinataires verront votre adresse « de ». Utiliser le même domaine d'expéditeur que le domaine De améliore la livraison. Rappelez-vous toujours, livrer des courriels avec succès dépend d'une chose - envoyer des courriels aux bonnes personnes de manière cohérente avec votre marque.

## Instructions étape par étape

Ajouter un expéditeur vérifié—connu comme l'adresse « De »—à vos campagnes marketing est une étape importante. Voici comment c'est fait :

<Stepper>

1. Cliquez sur le nom de votre entreprise/adresse courriel dans le coin supérieur droit (ceci ouvre un menu déroulant), puis cliquez sur Expéditeurs

1. Cliquez sur **+ Ajouter un expéditeur**
Après vérification, toute adresse courriel ajoutée à la liste d'expéditeurs peut être utilisée. Veuillez prendre un moment pour lire l'article suivant avant de changer votre adresse courriel d'expéditeur : Changer mon adresse courriel d'expéditeur.
**Rappel important :** Pour assurer une délivrabilité optimale et protéger votre réputation d'expéditeur, il est crucial d'envoyer des courriels depuis un domaine authentifié. Pour un guide détaillé sur la configuration de l'authentification de domaine, veuillez vous référer à notre guide d'authentification de domaine.
</Stepper>

## Dépannage

**Courriel de vérification d'expéditeur non reçu**

- Vérifiez les dossiers pourriels/courrier indésirable pour le courriel de vérification
- Assurez-vous que l'adresse courriel est tapée correctement sans espaces
- Attendez 10-15 minutes car les courriels peuvent être retardés
- Demandez un nouveau courriel de vérification depuis les paramètres d'expéditeur

**Impossible d'ajouter un courriel d'expéditeur des fournisseurs gratuits**

- Gmail, Yahoo et autres fournisseurs de courriel gratuits nécessitent l'authentification de domaine
- Utilisez une adresse courriel de domaine professionnel à la place
- Si vous devez utiliser un fournisseur gratuit, assurez-vous que votre compte a l'authentification appropriée
- Considérez mettre à niveau vers une solution de courriel d'entreprise

**Courriel d'expéditeur s'affichant comme « non vérifié »**

- Complétez le processus de vérification de courriel en cliquant sur le lien dans le courriel de vérification
- Vérifiez si le lien de vérification a expiré (habituellement 24-48 heures)
- Supprimez et re-ajoutez le courriel d'expéditeur si la vérification continue d'échouer
- Contactez le support si le courriel reste non vérifié après plusieurs tentatives

**Les destinataires voient un nom d'expéditeur différent de celui configuré**

- Videz le cache de votre client de courriel car il pourrait afficher d'anciennes informations d'expéditeur
- Vérifiez que le nom d'expéditeur ne contient pas de caractères spéciaux qui pourraient ne pas s'afficher correctement
- Vérifiez si les clients de courriel remplacent les noms d'affichage avec les informations de contact
- Assurez-vous d'utiliser le bon profil d'expéditeur pour votre campagne

**Impossible de changer l'adresse courriel d'expéditeur par défaut**

- Vérifiez que la nouvelle adresse courriel d'expéditeur est déjà ajoutée et vérifiée
- Vérifiez les permissions de compte si vous utilisez un compte d'équipe ou de revendeur
- Assurez-vous qu'aucune campagne active n'utilise l'expéditeur par défaut actuel
- Essayez de vous déconnecter et reconnecter pour rafraîchir les paramètres de compte
