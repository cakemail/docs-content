# Adresse d'expéditeur et langue par défaut

## Description

Les paramètres d'adresse d'expéditeur et de langue par défaut contrôlent comment tes courriels de bienvenue et de confirmation automatisés apparaissent aux nouveaux abonnés. Cette fonctionnalité te permet de personnaliser les préférences de langue pour tous les courriels générés par le système, incluant les confirmations d'abonnement, messages de bienvenue, et confirmations de désabonnement. Une configuration appropriée assure que ces points de contact automatisés s'alignent avec l'identité de ta marque et fournissent une expérience cohérente à travers toutes les interactions d'abonnés.

## Pourquoi c'est important
Pour bâtir ta marque et être reconnaissable aux destinataires, tu devrais choisir la bonne langue pour être reconnu par tes destinataires

## Instructions étape par étape

![Setting default sender and language](/images/forms/Nov-09-2021_11-11-00.gif)

Suis ces étapes simples pour spécifier correctement l'expéditeur par défaut pour ta liste et ses courriels de confirmation automatisés :

<Stepper>

1. Sélectionne l'onglet **Contacts**
![Select Contacts tab](/images/forms/B7680B8E-0822-4DE1-A29F-A3EEB9FE5DB8_4_5005_c.jpeg)

1. Choisis ta liste de contacts du menu déroulant
![Choose contact list](/images/forms/ACDED713-00D2-45DD-92E8-BA34863B7A27_4_5005_c.jpeg)

1. Clique sur l'**icône d'engrenage** pour accéder au menu de gestion de la liste
![Click gear icon](/images/forms/CD6FF683-4314-4B46-A481-B23A2F173071_4_5005_c.jpeg)

1. Sur ton menu de gestionnaire de liste, clique sur l'onglet **Paramètres**
![Click Settings tab](/images/forms/BA70BCD2-FF6F-4263-9E74-5E1179E5F04F_4_5005_c.jpeg)

1. Sélectionne ton expéditeur sur l'option **Expéditeur par défaut**
![Select default sender](/images/forms/04506F34-5070-40C0-A476-FD33B38C9515_4_5005_c.jpeg)
Tu peux ajouter ton courriel d'expéditeur si tu ne peux pas le trouver dans la liste en sélectionnant **Gérer les expéditeurs**.
*Référence : Ajouter un nouvel expéditeur*

1. Finalement, tu peux définir la langue par défaut de la liste à partir de ce menu. Sélectionne l'onglet **Paramètres** pour définir la **Langue par défaut** pour la liste
![Set default language](/images/forms/882BB14B-BD5B-4ACF-BACD-CB920E8E686A.png)
C'est la langue par défaut que les utilisateurs verront lors de la gestion de leurs abonnements. Ça n'affectera pas la langue de tes campagnes courriel.
</Stepper>

## Langues disponibles

Les langues supportées incluent :
- **Français (Canada)** - Pour les audiences québécoises
- **Français (France)** - Pour les audiences françaises
- **Anglais (Canada)** - Pour les audiences canadiennes anglophones
- **Anglais (États-Unis)** - Pour les audiences américaines
- **Espagnol** - Pour les audiences hispanophones

## Résolution de problèmes

**L'expéditeur par défaut ne s'affiche pas dans la liste**

- Assure-toi que l'adresse d'expéditeur est authentifiée
- Vérifie que tu as les permissions pour utiliser cette adresse
- Ajoute un nouvel expéditeur si nécessaire via "Gérer les expéditeurs"

**Les courriels automatisés sont dans la mauvaise langue**

- Vérifie la langue par défaut configurée pour la liste
- Assure-toi que la langue correspond à ton audience cible
- Les changements de langue s'appliquent seulement aux nouveaux abonnements

**Les abonnés ne reconnaissent pas l'expéditeur**

- Utilise une adresse d'expéditeur qui correspond à ta marque
- Évite les adresses noreply@ ou génériques
- Assure-toi que l'adresse d'expéditeur correspond à ton domaine

**Problèmes avec les caractères spéciaux**

- Vérifie que l'encodage UTF-8 est supporté
- Teste avec différents clients courriel
- Utilise des caractères standard si des problèmes persistent
