# Télécharger vers le gestionnaire de fichiers

## Description

Ta bibliothèque te permet de télécharger des fichiers comme des images ou des PDFs. Par exemple, tu peux stocker les images qui vont avec ta marque et structure établies. Tu peux utiliser ça de plusieurs façons à travers tes campagnes courriel.

## Pourquoi c'est important
Avoir toutes tes images et fichiers au même endroit t'aidera à te préparer à envoyer tes campagnes et les améliorer avec de super visuels ou du contenu utile.

## Instructions étape par étape

La procédure pour ajouter des fichiers au gestionnaire de fichiers est la suivante :

### Pour les images dans le contenu courriel :
<Stepper>

1. Sélectionne **Éditer courriel** dans la barre de campagne
![Campaign edit button location](/images/campaign-creation/A5E92D9F-D74F-48BA-A833-017C5A80467E_4_5005_c.jpeg)
![Edit email button in campaign bar](/images/campaign-creation/558DDDA4-0A8D-4A95-8086-0D375ABAA3FB_4_5005_c.jpeg)

1. Dans la section Aperçu courriel, sélectionne **Éditer courriel**
![Edit email button in preview section](/images/campaign-creation/C70DF271-09A8-4909-834F-2D307160106F_4_5005_c.jpeg)

1. Dans le menu Contenu, sélectionne **IMAGE** et glisse le bouton vers le contenu courriel
![Drag image content block to email](/images/campaign-creation/8F54E671-7BCF-42E9-A4B2-608DB413E3A8.png)

1. Clique **Parcourir**
![Browse button for image selection](/images/campaign-creation/04BF58E7-BBE8-450A-85CF-FB37F960B0D0_4_5005_c.jpeg)

1. Tu as maintenant trois options pour ajouter des fichiers :
   - **Télécharger** - Télécharge tes propres fichiers
   - **Importer** - Importe de sources externes
   - **Parcourir** - Parcours la sélection de photos gratuites.
![File upload options interface](/images/campaign-creation/D18A765E-5CA0-4C5A-9117-48B46310CB81_4_5005_c.jpeg)

1. Clique sur **Télécharger** et ensuite sélectionne ton fichier

1. Ton fichier est maintenant dans notre gestionnaire de fichiers et peut être sélectionné de là
**Note :** Actuellement, tu peux seulement télécharger un fichier à la fois (aucun FTP n'est disponible)
</Stepper>

### Pour les liens de fichiers :
<Stepper>

1. Clique le **bouton lien** dans la section où tu veux inclure ton fichier.

1. Utilise le menu déroulant du type d'URL, choisis **Gestionnaire de fichiers**

1. Sélectionne le fichier que tu as téléchargé plus tôt de ta bibliothèque.

1. Le fichier est maintenant lié à l'intérieur du courriel.

1. Les destinataires peuvent y accéder en cliquant sur le lien
</Stepper>

## Limitations de fichiers
- **Taille d'image maximale** : 1600px par 2000px
- **Taille de fichier maximale** : 1MB

## Résolution de problèmes

**Erreur de téléchargement échoué**

- Vérifie que la taille du fichier est sous la limite de 1MB
- Vérifie que les dimensions d'image ne dépassent pas 1600x2000px
- Assure-toi que le format de fichier est supporté (JPG, PNG, GIF, PDF)

**Le fichier n'apparaît pas dans la bibliothèque**

- Actualise la page du navigateur après le téléchargement
- Vide le cache du navigateur si des fichiers manquent
- Vérifie l'espace de stockage disponible dans ton compte

**Problèmes de qualité d'image**

- Optimise les images avant de télécharger (72-150 DPI pour le web)
- Utilise JPG pour les photos, PNG pour les graphiques avec transparence
- Compresse les gros fichiers sans perdre la qualité visible

**Impossible de sélectionner le fichier téléchargé**

- Attends que le téléchargement se termine complètement
- Vérifie les permissions et propriété du fichier
- Essaie de retélécharger si le fichier semble corrompu

**Limite de stockage atteinte**

- Supprime les fichiers non utilisés de la bibliothèque
- Compresse les fichiers avant de télécharger
- Contacte le support pour augmenter la capacité de stockage

**Vitesse de téléchargement lente**

- Vérifie la stabilité de la connexion internet
- Télécharge pendant les heures creuses
- Divise les gros téléchargements en plus petits lots
