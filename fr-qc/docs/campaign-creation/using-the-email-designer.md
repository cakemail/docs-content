# Utiliser le concepteur de courriel

<iframe src="//www.youtube-nocookie.com/embed/az96wcaWOnA" width="560" height="315" frameborder="0" allowfullscreen=""></iframe>

## Description

C'est l'outil d'édition principal que tu utiliseras pour créer un gabarit de courriel ; il est facile à utiliser et permet l'ajout et la suppression rapides de sections, la définition de différentes couleurs pour différentes zones, la décision sur la police et la largeur de ta campagne, les tests et bien plus, tout depuis une seule fenêtre.

## Pourquoi c'est utile

Cette fonction d'édition permet aux utilisateurs de créer des campagnes de courriel accrocheuses sans écrire de code pour accomplir n'importe quel objectif désiré. Avec cet outil, n'importe qui peut faire un excellent courriel. Le concepteur de courriel est maintenant plus flexible avec le déplacement de sections, la réorganisation de contenu, la fonctionnalité de sélecteur de couleur, et le versioning de contenu. Ici on va explorer tout ce qui concerne notre concepteur.

## Comment utiliser le concepteur de courriel

Voici quelques fonctionnalités de cet outil pour commencer et l'utiliser à son plein potentiel.

### Paramètres

Cette partie est pour définir le ton pour toute la campagne. Il y a beaucoup d'options à considérer :

- Largeur de zone de contenu (on recommande de ne pas dépasser 600 pixels)

- Alignement de zone de contenu. (Gauche ou Centre)

- Couleur d'arrière-plan

- Couleur d'arrière-plan de zone de contenu

- Police par défaut

- Couleur de lien

![Panneau de paramètres du concepteur de courriel montrant les couleurs d'arrière-plan, polices, et options de mise en page](/images/campaign-creation/campaigns-using-the-email-designer-312.png)

#### Ajoute ton pré-en-tête

Pour tirer le maximum de ta campagne, n'oublie pas d'ajouter un pré-en-tête convaincant. Fais simplement défiler vers le bas dans tes paramètres vers tes Propriétés optionnelles, et entre-le dans le champ pré-en-tête. C'est aussi simple que ça !

![Champ de texte pré-en-tête dans la section propriétés optionnelles](/images/campaign-creation/campaigns-using-the-email-designer-313.png)

Si tu as besoin d'un peu d'inspiration, notre fonctionnalité IA est là pour t'aider à générer des pré-en-têtes engageants qui résonnent avec ton audience. Essaie-la et trouve l'ajustement parfait pour ton message.

![Interface de génération de pré-en-tête alimentée par l'IA](/images/campaign-creation/campaigns-using-the-email-designer-314.png)

### Rangées

Avant d'ajouter du contenu à ta campagne, tu dois choisir une rangée. La section Rangées consiste en une variété de structures de composant. Sélectionne celle que tu veux et glisse-la n'importe où dans la campagne. Teste-les et trouve celle qui fonctionne le mieux pour ton gabarit.

![Options de mise en page de rangée montrant diverses configurations de colonnes](/images/campaign-creation/campaigns-using-the-email-designer-315.png)

![Glisser et déposer une mise en page de rangée dans le canevas de courriel](/images/campaign-creation/campaigns-using-the-email-designer-316.png)

#### Déplacer / Supprimer / Dupliquer la rangée

Repositionne, supprime ou duplique une section du courriel :

- D'abord, sélectionne la rangée que tu dois repositionner pour accéder à son menu de repositionnement.

Pour déplacer une section vers le haut ou le bas dans le gabarit :

- Dans le menu de positionnement, clique sur les flèches croisées trouvées à gauche pour déplacer ta rangée sélectionnée.

![Contrôles de positionnement de rangée avec flèches croisées pour déplacer les sections](/images/campaign-creation/campaigns-using-the-email-designer-317.png)

Pour supprimer une section :

- Dans le menu de positionnement, clique sur l'icône de poubelle à droite pour supprimer la rangée.

![Bouton de suppression de rangée avec icône de poubelle](/images/campaign-creation/campaigns-using-the-email-designer-318.png)

Pour dupliquer la rangée :

- Dans le menu de positionnement, clique sur le bouton de fenêtre double à droite, à côté de l'icône de poubelle.

![Bouton de duplication de rangée avec icône de fenêtre double](/images/campaign-creation/campaigns-using-the-email-designer-319.png)

#### Modifier

Plusieurs éléments peuvent être modifiés dans la section rangée. Sélectionne la rangée que tu souhaites modifier pour accéder à ce menu sur la partie droite du concepteur.

Voici les fonctionnalités que tu peux contrôler :

- Couleur d'arrière-plan de rangée

- Couleur d'arrière-plan de contenu

- Ne pas empiler sur mobile

- Image d'arrière-plan de rangée

- Personnaliser la colonne

- Condition d'affichage

- Cacher sur Bureau/Mobile

- Personnaliser les colonnes

- Arrière-plan de colonne

- Rembourrage

- Bordures

![Panneau de personnalisation de rangée montrant les paramètres de colonne, rembourrage, et bordures](/images/campaign-creation/campaigns-using-the-email-designer-320.png)

### Contenu

Ayant décidé quel type de rangées tu veux, tu peux maintenant choisir quel contenu inclure dans ces rangées.

Les options de contenu suivantes sont disponibles :

![Palette de blocs de contenu montrant tous les éléments disponibles comme Titre, Paragraphe, Image, Bouton, etc.](/images/campaign-creation/campaigns-using-the-email-designer-321.png)

- **Titre** : Utilise ceci pour insérer le titre de ta campagne.

- **Paragraphe** : Utilise ceci pour insérer un paragraphe de texte.

- **Liste** : Utilise ceci pour insérer une liste de contenu différent.

- **Image** : Utilise ceci pour insérer une image que tu as téléversée ou une image stock.

- **Bouton** : Utilise ceci pour insérer un bouton d'appel à l'action visible.

- **Diviseur** : Utilise ceci pour diviser ton contenu.

- **Espacement** : Ce bloc peut insérer de l'espace dans ton contenu de courriel.

- **Social** : Insère les icônes de médias sociaux et lie-les à tes pages.

- **HTML** : Insère ton code HTML personnalisé dans l'éditeur avec cette option.

- **Vidéo** : Insère une vignette montrant un aperçu de ta vidéo et un lien vers celle-ci.

- **Menu** : Utilise ceci pour inclure un menu interactif dans ton contenu de courriel.

- **GIFS** : Utilise ceci pour inclure un gif animé dans ton courriel et choisis parmi diverses options.

Chacun d'eux est modifiable dans son bloc. Glisse et dépose l'élément désiré dans ta section de contenu de courriel pour les utiliser. Pour créer la campagne parfaite, expérimente avec toutes les options dans le concepteur. Il n'y a presque aucune limite à ce que cet outil peut faire !

#### Revenir à une version précédente d'une campagne

Une version antérieure de ton courriel peut être restaurée. Dans le concepteur, clique sur l'icône « horloge » en bas à gauche de ta campagne. Tu peux alors revenir à une action spécifique dans l'édition.

![Icône d'horloge d'historique de version dans le coin inférieur gauche](/images/campaign-creation/campaigns-using-the-email-designer-322.png)

![Panneau d'historique de version montrant la liste des versions sauvegardées avec horodatages](/images/campaign-creation/campaigns-using-the-email-designer-323.png)

#### Mode aperçu

Tu peux prévisualiser le site en mode bureau et mobile en cliquant sur le bouton icône Œil trouvé en haut à droite du concepteur. Teste ton contenu sur les deux plateformes et assure-toi qu'il a l'air bien.

![Bouton de mode aperçu avec icône œil pour les vues bureau et mobile](/images/campaign-creation/campaigns-using-the-email-designer-324.png)

#### Sauvegarder les changements à la campagne.

![Bouton vert de sauvegarde pour préserver les changements de campagne](/images/campaign-creation/campaigns-using-the-email-designer-325.png)

Clique toujours sur le bouton vert Sauvegarder lors de la modification de ta campagne dans le concepteur de courriel. Il vaut mieux être prudent que désolé !

#### Sauvegarder la campagne comme gabarit, la réinitialiser ou l'abandonner

Si tu veux choisir une de ces options, clique sur le bouton flèche en haut à droite de l'écran. Tu peux en apprendre plus sur la sauvegarde de gabarits ici.

![Menu déroulant d'options supplémentaires avec Sauvegarder comme gabarit, Réinitialiser, et options d'abandon](/images/campaign-creation/campaigns-using-the-email-designer-326.png)

#### Détails de campagne

Si tu es satisfait de ton gabarit et souhaites l'envoyer, clique sur Continuer dans le coin supérieur droit du concepteur. La section Détails de campagne te fournit ces options pour envoyer la campagne.

Tu peux définir ce qui suit ici :

- Le nom de ta campagne

- La liste des abonnés

- Le segment dans cette audience,

- Le nom de l'expéditeur

- L'adresse courriel de l'expéditeur

- La ligne d'objet

Tu peux également revoir la campagne avec un test de courriel et programmer la livraison en sélectionnant Date et heure d'envoi.

![Formulaire de détails de campagne montrant les champs pour nom, liste, segment, info expéditeur, et ligne d'objet](/images/campaign-creation/campaigns-using-the-email-designer-327.png)

Rappelle-toi de la tester minutieusement avant de l'envoyer par courriel. Voici une liste de vérification de choses à faire avant d'envoyer une campagne et après qu'elle ait été envoyée.

## Dépannage

**Le concepteur de courriel ne se charge pas**

- Vide le cache et les témoins du navigateur
- Utilise Chrome ou Firefox pour la meilleure compatibilité
- Désactive les bloqueurs de publicité et extensions de navigateur

**Les changements ne se sauvegardent pas**

- Clique fréquemment sur le bouton vert Sauvegarder
- Vérifie la stabilité de la connexion Internet
- Cherche les messages de confirmation de sauvegarde automatique

**Problèmes d'aperçu mobile**

- Utilise des tests d'appareils réels pour la précision
- Vérifie les paramètres « Ne pas empiler sur mobile »
- Ajuste les largeurs de colonne pour l'affichage mobile

**Les images ne s'insèrent pas**

- Téléverse d'abord les images dans la bibliothèque
- Vérifie la taille du fichier image (max 1 Mo)
- Vérifie les dimensions d'image (max 1600x2000px)

**Le HTML personnalisé ne fonctionne pas**

- Valide le code HTML avant de l'insérer
- Supprime JavaScript (non supporté dans les courriels)
- Utilise du CSS en ligne au lieu de feuilles de style externes

**Problèmes d'historique de version**

- L'historique de version est limité aux modifications récentes
- Sauvegarde les versions importantes comme gabarits
- Ne peut pas récupérer les versions de plus de 30 jours
