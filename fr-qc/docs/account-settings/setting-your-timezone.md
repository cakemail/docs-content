# Configuration de ton fuseau horaire

## Description

Pour t'assurer que tes courriels sont programmés selon ton heure préférée, tu dois définir ton fuseau horaire avec précision. Ce guide va t'accompagner à travers les étapes pour changer ton fuseau horaire.

## Pourquoi c'est important

Le timing, c'est crucial dans le marketing par courriel. Assure-toi que tes courriels sont envoyés quand tu veux qu'ils soient envoyés - pas question de réveiller tes clients à 3h du matin!

## Instructions étape par étape

Accéder aux paramètres de ton compte

<Stepper>

1. Clique sur le nom de ton entreprise/adresse courriel dans le coin supérieur droit (ça ouvre un menu déroulant), puis clique sur Utilisateurs

1. Clique sur le compte utilisateur dont tu veux changer le fuseau horaire.

1. Ajuster le fuseau horaire
Sur la page utilisateur, localise l'option de fuseau horaire. Clique sur le menu déroulant et sélectionne ton fuseau horaire préféré parmi les options disponibles.

1. Sauvegarder les changements
Après avoir sélectionné le bon fuseau horaire, clique sur **Sauvegarder** pour appliquer les changements.
![Sauvegarder les changements de fuseau horaire](/images/accounts/accounts-setting-up-your-timezone-133.png)
Note que changer ton fuseau horaire affecte seulement l'heure de programmation de ton compte, pas le fuseau horaire du destinataire. Considère le fuseau horaire de ton public cible lors de la programmation des campagnes de courriel.
Définir ton fuseau horaire avec précision assure que tes campagnes de courriel sont programmées à ton heure préférée. En suivant ces étapes, tu peux ajuster ton fuseau horaire pour correspondre à ton emplacement et programmer les campagnes efficacement. Rappelle-toi de considérer le fuseau horaire de ton public cible pour un timing de livraison optimal.
</Stepper>

## Dépannage

**Les changements de fuseau horaire ne prennent pas effet**
- Sauvegarde les changements en cliquant sur le bouton « Sauvegarder »
- Déconnecte-toi et reconnecte-toi
- Vide le cache du navigateur
- Vérifie les campagnes programmées pour les mises à jour

**Impossible de trouver mon fuseau horaire**
- Cherche ta ville majeure la plus proche
- Utilise le décalage UTC si la ville n'est pas listée
- Vérifie les options d'heure d'été
- Contacte le support pour les fuseaux horaires manquants

**Les courriels programmés s'envoient au mauvais moment**
- Vérifie que ton paramètre de fuseau horaire est correct
- Vérifie les ajustements d'heure d'été
- Révise les paramètres de programmation de campagne
- Considère les fuseaux horaires des destinataires

**Plusieurs utilisateurs avec différents fuseaux horaires**
- Chaque utilisateur doit définir son propre fuseau horaire
- Le fuseau horaire au niveau du compte affecte les rapports
- La programmation de campagne utilise le fuseau horaire de l'expéditeur
- Coordonne les horaires d'équipe en conséquence

**Le fuseau horaire revient au paramètre précédent**
- Assure-toi d'avoir la permission de modifier les paramètres
- Vérifie l'interférence du remplissage automatique du navigateur
- Essaie d'utiliser un navigateur différent
- Contacte le support si le problème persiste
