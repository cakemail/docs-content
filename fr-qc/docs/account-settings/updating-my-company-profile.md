# Mise à jour de mon profil d'entreprise

## Description

Les paramètres de ton profil d'entreprise doivent être à jour avant d'envoyer ta première campagne, car la loi exige une adresse physique dans chaque campagne.

## Pourquoi c'est important

- **Conformité légale** : L'adresse physique est requise dans chaque campagne par la loi
- **Cohérence de marque** : Le logo de ton entreprise peut être utilisé dans tes campagnes de courriel
- **Apparence professionnelle** : Les informations complètes du profil inspirent confiance
- **Informations de contact** : Fournit aux destinataires des détails d'entreprise légitimes

## Instructions étape par étape

Pour mettre à jour le profil de ton entreprise, suis les instructions.

<Stepper>

1. Clique sur le nom de ton entreprise/adresse courriel dans le coin supérieur droit (ça ouvre un menu déroulant), puis clique sur Compte

1. Clique sur le **bouton crayon** pour modifier chaque partie
</Stepper>

## Champs que tu peux ajouter/modifier :

Voici une liste des champs que tu peux ajouter/modifier :

- **Logo de l'entreprise** (peut être utilisé dans tes courriels)
- **Nom légal de l'entreprise**
- **Adresse** 
- **Numéro de téléphone**
- **Site web**

## Meilleures pratiques

- **Garde les informations à jour** : Mets à jour les détails chaque fois que les informations de ton entreprise changent
- **Complète tous les champs** : Remplis tous les champs pertinents pour une apparence professionnelle
- **Logo de haute qualité** : Utilise une image de logo claire et professionnelle
- **Adresse exacte** : Assure-toi que ton adresse physique est correcte et actuelle
- **Informations de contact valides** : Fournis des numéros de téléphone qui fonctionnent et des URL de site web

## Dépannage

**Impossible de télécharger le logo de l'entreprise**

- Vérifie la taille du fichier (max 5 Mo)
- Utilise le format PNG ou JPG
- Assure-toi que les dimensions de l'image sont raisonnables
- Essaie de réduire la résolution de l'image

**Erreurs de validation d'adresse**
- Inclus tous les champs requis
- Utilise le format d'adresse standard
- Évite les caractères spéciaux
- Vérifie le format du code postal

**Les changements ne se sauvegardent pas**
- Clique sur « Sauvegarder » après chaque modification
- Vérifie les erreurs de validation
- Assure-toi que tous les champs requis sont remplis
- Essaie de modifier un champ à la fois

**Le logo n'apparaît pas dans les courriels**
- Publie les changements après le téléchargement
- Vérifie que le gabarit de courriel inclut la balise de fusion du logo
- Vérifie que l'URL du logo est accessible
- Vide le cache du client de courriel

**Impossible de modifier les informations de l'entreprise**
- Confirme que tu as les privilèges d'administrateur
- Vérifie si le compte est suspendu
- Assure-toi que les champs ne sont pas verrouillés par l'organisation
- Contacte le propriétaire du compte pour les permissions

**URL du site web non acceptée**
- Inclus l'URL complète (https://...)
- Supprime les espaces de fin
- Assure-toi que l'URL est publiquement accessible
- Utilise le format de domaine standard
