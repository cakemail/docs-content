# Authentification multi-facteurs (AMF)

## Description

L'authentification multi-facteurs (AMF) est une mesure de sécurité qui renforce la protection de ton compte. En activant l'AMF, tu améliores activement la défense de ton compte contre l'accès non autorisé. Ce guide te fournit des instructions étape par étape pour activer l'AMF sur ton compte.

## Pourquoi c'est important

Dans le monde numérique d'aujourd'hui, protéger tes comptes en ligne, c'est essentiel - surtout si tu gères une entreprise à Montréal, Québec ou ailleurs au Québec. L'authentification multi-facteurs (AMF) fournit une couche supplémentaire de sécurité, réduisant le risque d'accès non autorisé et de violations potentielles de données. Mettre en place l'AMF, c'est une étape proactive vers le renforcement de la sécurité de ton compte - pense-y comme à avoir une serrure Medeco sur ta porte plutôt qu'une serrure ordinaire!

Comme dirait Maurice dans La Petite Vie : « Y faut que ça soye sécure ! » Et c'est exactement ça avec l'AMF - on veut que ça soye full sécure pour ton compte.

## Instructions étape par étape

<Stepper>

1. Clique sur le nom de ton entreprise/adresse courriel dans le coin supérieur droit (ça ouvre un menu déroulant), puis clique sur Utilisateurs

1. Choisis l'utilisateur pour l'activation de l'AMF

1. Fais défiler vers le bas jusqu'à « Sécurité » et clique sur « Activer » pour activer la vérification en 2 étapes
![Activer la vérification en 2 étapes](/images/accounts/accounts-setting-up-multi-factor-authentication-mfa-for-your-account-155.png)
</Stepper>

## Inscription avec l'application d'authentification

Les utilisateurs peuvent s'inscrire en utilisant leur application d'authentification en scannant le code QR avec leur appareil mobile.

![Scanner le code QR avec l'application d'authentification](/images/accounts/accounts-setting-up-multi-factor-authentication-mfa-for-your-account-156.png)

Entre le code à 6 chiffres affiché dans ton application d'authentification pour compléter la configuration de l'AMF.

![Entrer le code de vérification à 6 chiffres](/images/accounts/accounts-setting-up-multi-factor-authentication-mfa-for-your-account-157.png)

## Codes de sauvegarde pour une sécurité supplémentaire

Si tu perds l'accès à ton appareil mobile (ça arrive même aux meilleurs, surtout quand on perd son cell dans un banc de neige en hiver !), tu peux utiliser l'un des 10 codes de sécurité de sauvegarde. Ces codes servent de sécurité de secours, accordant l'accès quand ton appareil principal n'est pas disponible. Stocke ces codes de manière sécurisée - pas juste sur un Post-it collé sur ton écran !

**Conseil de pro :** Garde une copie dans ton portefeuille et une autre dans ton bureau. C'est comme avoir une clé de spare chez ta mère - on sait jamais quand on en aura besoin.

## Indépendance pour personnaliser

Avec l'authentification multi-facteurs (AMF), chaque utilisateur de compte peut indépendamment activer ou désactiver l'AMF selon ses préférences et besoins.

## Dépannage

**Impossible de scanner le code QR**
- Assure-toi que les permissions de caméra sont activées
- Augmente la luminosité de l'écran (surtout si tu es dans un Tim Hortons avec un éclairage ordinaire)
- Utilise l'option de saisie manuelle si disponible
- Essaie une application d'authentification différente - Google Authenticator ou Microsoft Authenticator marchent full bien

**Le code à 6 chiffres ne fonctionne pas**
- Vérifie que l'heure de ton appareil est synchronisée
- Entre le code rapidement (expire en 30 secondes - c'est plus rapide qu'un changement de ligne dans le métro de Montréal !)
- Assure-toi d'utiliser le bon compte dans l'application
- Essaie de générer un nouveau code - ça prend juste quelques secondes

**Perte d'accès à l'application d'authentification**
- Utilise l'un de tes 10 codes de sauvegarde
- Contacte le support avec la vérification du compte
- Aie les informations de récupération du compte prêtes
- Considère stocker les codes de sauvegarde de manière sécurisée

**Les codes de sauvegarde ne fonctionnent pas**
- Assure-toi d'entrer le code correctement
- Chaque code de sauvegarde ne peut être utilisé qu'une seule fois
- Vérifie si le code a déjà été utilisé
- Contacte le support si tous les codes échouent

**Tu veux désactiver l'AMF**
- Va aux paramètres de sécurité dans ton profil
- Clique sur « Désactiver » à côté de la vérification en 2 étapes
- Confirme avec ton mot de passe
- Considère les implications de sécurité avant de désactiver
