## 🎨 Principes de design

### 1. **mobile en premier**
- 50%+ des courriels sont ouverts sur mobile
- Les layouts à une colonne marchent le mieux
- Police minimum 14px
- Zones cliquables minimum 44px

### 2. **Hiérarchie visuelle**
- Structure de titres claire
- Blocs de contenu scannables
- Utilisation stratégique de l'espace blanc
- Un call-to-action principal

### 3. **Accessibilité**
- Texte alternatif pour toutes les images
- Contraste de couleur suffisant (4.5:1)
- Ordre de lecture logique
- Compatible avec les lecteurs d'écran

## 💡 Guidelines essentielles

### Meilleures pratiques de layout
- **Largeur** : 600-640px maximum
- **Pré-en-tête** : 40-90 caractères
- **Au-dessus du pli** : message clé et CTA
- **Structure** : pyramide inversée (important → détails)

### Typographie
- **Polices** : polices web-safe (Arial, Georgia, Times)
- **Tailles** : 14-16px pour le corps, 22-28px pour les titres
- **Interligne** : 1.5x pour la lisibilité
- **Couleurs** : contraste élevé pour bien lire

### Images
- **Format** : JPEG pour photos, PNG pour graphiques
- **Taille** : optimise pour le web (moins de 1 MB total)
- **Ratio** : 60% texte, 40% images
- **Affichage** : toujours inclure du texte alternatif

## 📊 Métriques de performance

Des courriels bien designés obtiennent :
- **Meilleurs taux d'ouverture** : aperçu accrocheur
- **Meilleurs taux de clics** : CTAs clairs
- **Moins de désabonnements** : apparence professionnelle
- **Plus de conversions** : design user-friendly

## 🚀 Processus de design

### 1. **Planifie** (10 minutes)
- Définis l'objectif du courriel
- Identifie le message clé
- Choisis un gabarit

### 2. **Design** (30 minutes)
- Crée une hiérarchie visuelle
- Écris du copy accrocheur
- Ajoute des images stratégiquement

### 3. **Teste** (20 minutes)
- Preview sur plusieurs appareils
- Check le rendu dans différents clients
- Vérifie que tous les liens marchent

### 4. **Optimise** (continu)
- Teste A/B des éléments de design
- Analyse les données de performance
- Itère selon les résultats

## ⚠️ Erreurs courantes à éviter

- Utiliser trop de polices ou couleurs
- Négliger l'optimisation mobile
- Créer des courriels tout-en-image
- Cacher du texte important dans les images
- Oublier le texte alternatif
- Mauvais contraste de couleurs
- Layouts trop chargés

## 🛠️ Outils et ressources

### Test de courriels
- Preview dans plusieurs clients
- Check ton score spam
- Vérifie le rendu

### Optimisation d'images
- Compresse sans perdre la qualité
- Utilise les bons formats
- Implémente des images responsives

### Vérification d'accessibilité
- Analyseurs de contraste
- Test avec lecteur d'écran
- Vérification du texte alternatif
