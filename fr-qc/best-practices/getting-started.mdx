4. **Référence** : garde le [Glossaire](/fr-qc/best-practices/getting-started/glossary-of-email-marketing-terms) proche pour les termes que tu connais pas

## 💡 Tips rapides pour débuter

- **Commence petit** : un simple courriel de bienvenue avant les campagnes complexes
- **Qualité > quantité** : focus sur des abonnés engagés, pas juste un gros nombre
- **Teste tout** : envoie toujours des tests avant de lancer pour vrai
- **Mobile first** : plus de 50% des courriels sont ouverts sur mobile
- **Sois constant** : envoyer régulièrement crée des attentes chez ton monde

## 📊 Métriques de succès pour débuter

Focus sur ces chiffres au début :
- **Taux d'ouverture** : 15-25% c'est un bon début
- **Taux de clic** : 2-5% = contenu engageant
- **Taux de désabonnement** : reste en bas de 0,5%
- **Taux de bounce** : doit être moins de 2%

## Le mindset du débutant

### Ce qui est normal :
- Faire des erreurs (tout le monde en fait)
- Avoir des taux moyens au début
- Prendre du temps à trouver ta voix
- Tester plein d'affaires

### Ce qui est important :
- Respecter tes abonnés
- Rester authentique
- Apprendre de tes métriques
- Être patient

## Les erreurs classiques à éviter

1. **Envoyer à tout le monde** : segmente dès le début
2. **Ignorer le mobile** : toujours tester sur téléphone
3. **Être trop vendeur** : 80% valeur, 20% vente
4. **Négliger les tests** : test, test, test !
5. **Acheter des listes** : jamais. Point final.

## Ressources essentielles

- **Templates de base** : courriel de bienvenue, infolettre, promo
- **Outils gratuits** : Litmus pour preview, Grammarly pour l'orthographe
- **Inspiration** : Really Good Emails, Milled
- **Formation** : HubSpot Academy, Mailchimp Resources
