## 🎯 Comment bâtir ta stratégie

### 1. **Les fondations** (Semaine 1-2)
- Setup des bons formulaires d'inscription
- Créer une proposition de valeur solide
- Activer le double opt-in

### 2. **La croissance** (Mois 1-3)
- Lancer tes lead magnets
- Optimiser tes points de conversion
- Tester différents incitatifs

### 3. **L'entretien** (Tout le temps)
- Nettoyer ta liste régulièrement
- Campagnes de réengagement
- Surveiller la santé de ta liste

## 💡 Les principes de base

### Qualité > Quantité
- 1000 abonnés engagés valent mieux que 10000 fantômes
- Focus sur ton audience idéale
- Meilleur engagement = meilleure délivrabilité

### La permission, c'est sacré
- Toujours utiliser l'opt-in explicite
- Jamais acheter ou louer de listes
- Rendre le désabonnement facile

### L'échange de valeur
- Offre quelque chose de solide pour avoir leur courriel
- Sois clair sur ce qu'ils vont recevoir
- Livre ce que tu promets, toujours

## 📊 Les métriques de santé

Vérifie ça à chaque mois :
- **Taux de croissance** : nouveaux abonnés - désabonnements
- **Taux d'engagement** : abonnés actifs / liste totale
- **Taux de bounce** : doit être moins de 2%
- **Churn de liste** : dégradation annuelle (environ 22,5%)

## 🚀 Les tactiques qui marchent

1. **Content upgrades** : du bonus pour ceux qui lisent ton blog
2. **Pop-ups de sortie** : attraper ceux qui s'en vont
3. **Médias sociaux** : promouvoir tes incitatifs
4. **Programme de référence** : tes abonnés qui partagent
5. **Webinaires** : du contenu de valeur = inscriptions

## ⚠️ Les erreurs classiques

À éviter comme la peste :
- Pré-cocher les cases d'inscription
- Cacher le lien de désabonnement
- Acheter ou louer des listes
- Pas nettoyer ta liste
- Ignorer les inactifs
- Ajouter du monde sans permission
